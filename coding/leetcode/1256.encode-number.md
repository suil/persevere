# Encode Number
[1256. Encode Number](https://leetcode.com/problems/encode-number/)

Given a non-negative integer num, Return its encoding string.

The encoding is done by converting the integer to a string using a secret function that you should deduce from the following table:



 
Example 1:


Input: num = 23
Output: "1000"


Example 2:


Input: num = 107
Output: "101100"


 
Constraints:


	0 <= num <= 10^9

# Key insights
- The encoding follows the pattern where encode(num) is the binary representation of (num + 1) with the leading '1' bit removed
- Converting num+1 to binary and slicing off the first character produces the encoded string
- This leverages the observation that encoded values map to positions in a binary tree structure

# Code
```javascript
/**
 * @param {number} num
 * @return {string}
 */
var encode = function(num) {
    return (num + 1).toString(2).slice(1);
};

/*
23 / 8 = 2, 7

107 / 8 
*/
```

# Complexity
**Time:** O(log n) -- binary conversion of n+1
**Space:** O(log n) -- for the binary string
