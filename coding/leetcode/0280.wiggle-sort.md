# Wiggle Sort
[280. Wiggle Sort](https://leetcode.com/problems/wiggle-sort/)

Given an integer array nums, reorder it such that nums[0] <= nums[1] >= nums[2] <= nums[3]....

# Key insights
- Single pass through the array, checking adjacent pairs
- At even indices: ensure nums[i] <= nums[i+1], swap if violated
- At odd indices: ensure nums[i] >= nums[i+1], swap if violated
- Swapping maintains previous conditions because we're only enforcing local constraints

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var wiggleSort = function(nums) {
    for (let i = 0; i < nums.length - 1; i++) {
        if (i + 1 < nums.length && i % 2 === 0 && nums[i] > nums[i + 1]) {
            [nums[i], nums[i + 1]] = [nums[i + 1], nums[i]];
        }
        if (i + 1 < nums.length && i % 2 === 1 && nums[i] < nums[i + 1]) {
            [nums[i], nums[i + 1]] = [nums[i + 1], nums[i]];
        }
    }
    return nums;
};
```

# Complexity
**Time:** O(n) - single pass through the array
**Space:** O(1) - in-place swaps, no extra space
