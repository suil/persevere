# Range Sum Query 2D - Immutable
[304. Range Sum Query 2D - Immutable](https://leetcode.com/problems/range-sum-query-2d-immutable/)

# Key insights
- Build a 2D prefix sum matrix where dp[i][j] = sum of rectangle from (0,0) to (i-1,j-1)
- Prefix sum formula: dp[i+1][j+1] = dp[i+1][j] + dp[i][j+1] - dp[i][j] + matrix[i][j]
- Query uses inclusion-exclusion: subtract top and left rectangles, add back the overlap

# Code
```javascript
/**
 * @param {number[][]} matrix
 */
var NumMatrix = function(matrix) {
    if (matrix == null || matrix.length === 0 || matrix[0].length === 0) return;

    const rowLen = matrix.length;
    const colLen = matrix[0].length;

    const dp = [...Array(rowLen + 1)].map(_ => [...Array(colLen + 1)].fill(0));
    for (let row = 0; row < rowLen; row++) {
        for (let col = 0; col < colLen; col++) {
            dp[row + 1][col + 1] = dp[row + 1][col] + dp[row][col + 1] + matrix[row][col] - dp[row][col];
        }
    }
    this.dp = dp;
};

/** 
 * @param {number} row1 
 * @param {number} col1 
 * @param {number} row2 
 * @param {number} col2
 * @return {number}
 */
NumMatrix.prototype.sumRegion = function(row1, col1, row2, col2) {
    return this.dp[row2 + 1][col2 + 1]
        - this.dp[row1][col2 + 1]
        - this.dp[row2 + 1][col1]
        + this.dp[row1][col1];
};

/** 
 * Your NumMatrix object will be instantiated and called as such:
 * var obj = new NumMatrix(matrix)
 * var param_1 = obj.sumRegion(row1,col1,row2,col2)
 */
```

# Complexity
**Time:** O(m * n) for construction, O(1) for each sumRegion query
**Space:** O(m * n) - for the prefix sum matrix
