# Sum of All Subset XOR Totals
[1863. Sum of All Subset XOR Totals](https://leetcode.com/problems/sum-of-all-subset-xor-totals/)

The XOR total of an array is defined as the bitwise XOR of all its elements, or 0 if the array is empty.

# Key insights
- Each element has two choices: include in subset (XOR it) or exclude
- Backtracking explores all 2^n subsets by branching at each element
- Base case returns the XOR total when all elements are processed
- Sum both branches: include current element vs exclude it

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var subsetXORSum = function(nums) {
    
    function backtrack(i, total) {
        if (i === nums.length) return total;
        return backtrack(i + 1, total ^ nums[i]) + backtrack(i + 1, total);
    }

    return backtrack(0, 0);
};
```

# Complexity
**Time:** O(2^n) - exploring all possible subsets
**Space:** O(n) - recursion stack depth
