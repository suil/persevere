# Read N Characters Given Read4
[157. Read N Characters Given Read4](https://leetcode.com/problems/read-n-characters-given-read4/)

Given a file and assume that you can only read the file using a given method read4, implement a method to read n characters.

# Key insights
- Repeatedly call read4 into a temporary buffer, copying min(count, n-index) characters to the destination buffer
- Stop when read4 returns fewer than 4 characters (EOF) or the destination buffer is full (index === n)
- The count is clamped to avoid writing past the requested n characters

# Code
```javascript
/**
 * Definition for read4()
 * 
 * @param {character[]} buf4 Destination buffer
 * @return {number} The number of actual characters read
 * read4 = function(buf4) {
 *     ...
 * };
 */

/**
 * @param {function} read4()
 * @return {function}
 */
var solution = function(read4) {
    /**
     * @param {character[]} buf Destination buffer
     * @param {number} n Number of characters to read
     * @return {number} The number of actual characters read
     */
    return function(buf, n) {
        const buf4 = [];
        let index = 0;

        while (true) {
            let count = read4(buf4);
            count = Math.min(count, n - index)
            
            for (let i = 0; i < count; i++) {
                buf[index++] = buf4[i];
            }

            if (count < 4 || index === n) return index;
        }
    };
};
```

# Complexity
**Time:** O(n)
**Space:** O(1) -- only a fixed-size buf4 of length 4
