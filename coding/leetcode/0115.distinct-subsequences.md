# Distinct Subsequences
[115. Distinct Subsequences](https://leetcode.com/problems/distinct-subsequences/)

Given two strings s and t, return the number of distinct subsequences of s which equals t.

# Key insights
- 2D DP where dp[i][j] = number of ways to form t[0..j-1] from s[0..i-1]
- If s[i-1] === t[j-1], dp[i][j] = dp[i-1][j-1] (use the match) + dp[i-1][j] (skip s[i-1])
- If characters differ, dp[i][j] = dp[i-1][j] (skip s[i-1]); base case dp[i][0] = 1 (empty t)

```javascript
    // return backtrack(0, 0);

    const dp = Array(s.length + 1).fill().map(() => Array(t.length + 1).fill(0));

    for (let i = 0; i <= s.length; i++) {
        dp[i][0] = 1;
    }

    for (let i = 1; i <= s.length; i++) {
        for (let j = 1; j <= s.length; j++) {
            if (s[i - 1] === t[j - 1]) {
                dp[i][j] = dp[i - 1][j] + dp[i - 1][j - 1];
            } else {
                dp[i][j] = dp[i - 1][j];
            }
        }
    }
    return dp[s.length][t.length];
};
```

# Complexity
**Time:** O(s.length * t.length)
**Space:** O(s.length * t.length)
