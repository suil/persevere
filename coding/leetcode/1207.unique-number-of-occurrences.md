# Unique Number of Occurrences
[1207. Unique Number of Occurrences](https://leetcode.com/problems/unique-number-of-occurrences/)

Given an array of integers arr, return true if the number of occurrences of each value in the array is unique or false otherwise.

# Key insights
- Counts occurrences of each value using a Map
- Places all occurrence counts into a Set; if the Set size equals the Map size, all counts are unique
- A duplicate count would reduce the Set size below the Map size

# Code
```javascript
/**
 * @param {number[]} arr
 * @return {boolean}
 */
var uniqueOccurrences = function(arr) {
    const map = new Map();
    for (const item of arr) {
        map.set(item, (map.get(item) || 0) + 1);
    }
    const set = new Set([...map.values()]); 
    return set.size === map.size;
};
```

# Complexity
**Time:** O(n) -- single pass to count, then O(n) to build the set
**Space:** O(n) -- for the frequency map and set
