# Find Pivot Index
[724. Find Pivot Index](https://leetcode.com/problems/find-pivot-index/)

Given an array of integers nums, calculate the pivot index of this array.

# Key insights
- The pivot condition (left sum == right sum) is equivalent to checking if prefixSum == (totalSum - nums[i]) / 2
- Compute total sum first, then maintain a running prefix sum to check the condition in a single pass
- No need to compute right sum separately since it equals totalSum - prefixSum - nums[i]

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var pivotIndex = function(nums) {
    const sum = nums.reduce((s, n) => s + n, 0);
    let prefixSum = 0;
    for (let i = 0; i < nums.length; i++) {
        if ((sum - nums[i]) / 2 === prefixSum) {
            return i;
        }
        prefixSum += nums[i];
    }
    return -1;
};
```

# Complexity
**Time:** O(n)
**Space:** O(1)
