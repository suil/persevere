# Add Strings
[415. Add Strings](https://leetcode.com/problems/add-strings/)

Given two non-negative integers, num1 and num2 represented as string, return the sum of num1 and num2 as a string.

# Key insights
- Simulate addition from right to left, like manual addition
- Track carry for when digit sum >= 10
- Convert characters to digits using character codes
- Don't forget to add remaining carry at the end

# Code
```javascript
/**
 * @param {string} num1
 * @param {string} num2
 * @return {string}
 */
var addStrings = function(num1, num2) {
    let index1 = num1.length - 1;
    let index2 = num2.length - 1;
    let carry = 0;
    let output = [];
    const base = '0'.charCodeAt(0);
    
    while (index1 >= 0 || index2 >= 0) {
        let sum = carry;
        if (index1 >= 0) {
            sum += num1.charCodeAt(index1--) - base;
        }
        if (index2 >= 0) {
            sum += num2.charCodeAt(index2--) - base;
        }
        output.unshift(sum % 10);
        carry = Math.floor(sum / 10);
    }
    
    if (carry > 0) {
        output.unshift(carry);
    }
    
    return output.join('');
};
```

# Complexity
**Time:** O(max(n, m)) - where n and m are lengths of the two strings
**Space:** O(max(n, m)) - for the output array
