# Minimum Absolute Difference in BST
[530. Minimum Absolute Difference in BST](https://leetcode.com/problems/minimum-absolute-difference-in-bst/)

Given the root of a Binary Search Tree (BST), return the minimum absolute difference between the values of any two different nodes in the tree.

Example 1:
        4
     /     \
    2       6
   / \
  1   3

Input: root = [4,2,6,1,3]
Output: 1

Example 2:

        1
     /     \
    0       48
   / \
 12   49

Input: root = [1,0,48,null,null,12,49]
Output: 1

# Key insights
* In-order traversal of BST visits nodes in sorted order
* Compare each node with previous node to find minimum difference
* Single pass traversal with O(1) space using recursion stack
* BST property eliminates need to compare all node pairs

# Code
```javascript
var getMinimumDifference = function(root) {
    let minDiff = Infinity;
    let prevVal;
    
    function inorder(node) {
        if (!node) {
            return;
        }
        
        inorder(node.left);
        
        if (prevVal !== undefined) {
            minDiff = Math.min(minDiff, Math.abs(node.val - prevVal));
        }
        prevVal = node.val;
        
        inorder(node.right);
    }
    
    inorder(root);
    return minDiff;
};
```
