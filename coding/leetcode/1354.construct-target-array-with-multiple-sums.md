# Construct Target Array With Multiple Sums
[1354. Construct Target Array With Multiple Sums](https://leetcode.com/problems/construct-target-array-with-multiple-sums/)

You are given an array target of n integers. From a starting array arr consisting of n 1's, you may perform the following procedure :

# Key insights
- Works backwards from the target array: the largest element must have been the result of the last operation
- Uses a max priority queue to repeatedly extract the largest element and reverse-compute its previous value (x = max % restSum)
- Uses modulo instead of repeated subtraction to handle large values efficiently, avoiding TLE

# Code
```javascript
/**
 * @param {number[]} target
 * @return {boolean}
 */
var isPossible = function(target) {
    const n = target.length;
    if (n === 1) {
        if (target[0] === 1) {
            return true;
        }
        return false;
    }
    let count = 0, sum = 0;
    const q = new MaxPriorityQueue();
    for (let i = 0; i < n; i++) {
        if (target[i] === 1) {
            count++;
        }
        q.enqueue(target[i]);
        sum += target[i];
    }
    while (count < n) {
        const temp = q.dequeue();
        sum -= temp;
        if (temp - sum <= 0) {
            return false;
        }
        let x = temp % sum;
        if (x === 0) {
            x = sum;
        }
        if (x === 1) {
            count++;
        }
        sum += x;
        q.enqueue(x);
    }
    return true;
};

```

# Complexity
**Time:** O(n log n + log(maxVal) * log n) -- initial heap build plus repeated extract/insert operations
**Space:** O(n) -- for the priority queue
