# Top K Frequent Words
[692. Top K Frequent Words](https://leetcode.com/problems/top-k-frequent-words/)

Given an array of strings words and an integer k, return the k most frequent strings.

# Key insights
- Count word frequencies with a map, then use a min-heap of size k to efficiently find the top k elements
- Custom comparator: lower frequency comes first (min-heap), with reverse lexicographic order for ties so that lexicographically smaller words survive eviction
- Extract results from the heap in reverse order (unshift) to get descending frequency / ascending lexicographic order

# Code
```javascript
/**
 * @param {string[]} words
 * @param {number} k
 * @return {string[]}
 */
var topKFrequent = function(words, k) {
    const map = new Map();
    const minQ = new PriorityQueue((a, b) => {
        if (a[0] === b[0]) {
            return b[1].localeCompare(a[1]);
        }
        return a[0] - b[0];
    });

    for (const word of words) {
        map.set(word, (map.get(word) || 0) + 1);
    }

    for (const [word, count] of map) {
        minQ.enqueue([count, word]);
        if (minQ.size() > k) minQ.dequeue();
    }

    const result = [];
    while (!minQ.isEmpty()) {
        result.unshift(minQ.dequeue()[1]);
    }
    return result;
};
```

# Complexity
**Time:** O(n log k) where n is the number of words -- heap operations for each unique word
**Space:** O(n) -- for the frequency map and heap
