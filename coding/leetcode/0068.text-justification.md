# Text Justification
[68. Text Justification](https://leetcode.com/problems/text-justification/)

Given an array of strings words and a width maxWidth, format the text such that each line has exactly maxWidth characters and is fully (left and right) justified.

# Key insights
- First pass: greedily pack words into rows, tracking current line length including single spaces
- Second pass: for each row, distribute extra spaces evenly across gaps; leftmost gaps get one extra space when remainder exists
- Last line and single-word lines are left-justified with trailing spaces

# Code
```javascript
/**
 * @param {string[]} words
 * @param {number} maxWidth
 * @return {string[]}
 */
var fullJustify = function(words, maxWidth) {
    const rows = [[]];
    let currLineLen = 0;

    for (const word of words) {
        const wordLen = word.length + (currLineLen === 0 ? 0 : 1);
        if (currLineLen + wordLen <= maxWidth) { // same row
            rows.at(-1).push(word);
            currLineLen += wordLen;
        } else { // next row
            rows.push([word]);
            currLineLen = word.length;
        }
    }

    const res = [];

    for (let row = 0; row < rows.length; row++) {
        const rowWords = rows[row];
        let line = '';
        if (rowWords.length === 1 || row === rows.length - 1) {
            line += rowWords.join(' ');
            line += ' '.repeat(maxWidth - line.length)
            res.push(line)
            continue;
        }

        const wordLen = rowWords.reduce((sum, w) => sum + w.length, 0);
        const gapCount = rowWords.length - 1;
        const gapSpaceCount = Math.floor((maxWidth - wordLen) / gapCount);
        let extraSpaces = (maxWidth - wordLen) % gapCount;

        for (let i = 0; i < rowWords.length; i++) {
            let spaces = '';
            if (i > 0) {
                spaces = ' '.repeat(gapSpaceCount + (extraSpaces > 0 ? 1 : 0));
                extraSpaces--;
            }
            line += spaces + rowWords[i];
        }
        res.push(line)
    }

    return res;
};
```

# Complexity
**Time:** O(n) where n is the total number of characters across all words
**Space:** O(n) for the output
