# Valid Palindrome II
[680. Valid Palindrome II](https://leetcode.com/problems/valid-palindrome-ii/)

Given a string s, return true if the s can be palindrome after deleting at most one character from it.

# Key insights
- Two-pointer approach from both ends; when a mismatch is found, try skipping either the left or right character and check if the remaining substring is a palindrome
- At most one deletion is allowed, so only two sub-checks are needed at the first mismatch point
- Early return if the string is already a palindrome

# Code
```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var validPalindrome = function(s) {
    
    function isPalindrome(s, left, right) {
        while (left < right) {
            if (s[left] !== s[right]) return false;
            left++;
            right--;
        }
        return true;
    }

    if (isPalindrome(s, 0, s.length - 1)) {
        return true;
    }

    let left = 0;
    let right = s.length - 1;
    
    while (left < right) {
        if (s[left] === s[right]) {
            left++;
            right--;
        } else {
            if (isPalindrome(s, left, right - 1) || isPalindrome(s, left + 1, right)) {
                return true;
            }
            return false;
        }
    }
    return true;
};
```

# Complexity
**Time:** O(n) -- at most two linear scans of the string
**Space:** O(1)
