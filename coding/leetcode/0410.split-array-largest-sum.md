# Split Array Largest Sum
[410. Split Array Largest Sum](https://leetcode.com/problems/split-array-largest-sum/)

Given an integer array nums and an integer k, split nums into k non-empty subarrays such that the largest sum of any subarray is minimized.

# Key insights
- Binary search on the answer (the maximum subarray sum)
- Search range: [max element, sum of all elements]
- For each candidate limit, greedily check if we can split into k or fewer subarrays
- Start a new subarray when adding current element would exceed the limit

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var splitArray = function(nums, k) {
    let left = Math.max(...nums);
    let right = nums.reduce((s, n) => s + n, 0);

    function canSplit(splitLimit) {
        let sum = 0;
        let count = 1;
        for (const num of nums) {
            if (sum + num > splitLimit) { // boundary
                count++;
                sum = num;
            } else {
                sum += num;
            }
        }
        return count <= k; // needs to be less than
    }

    let res = right;
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (canSplit(mid)) {
            right = mid - 1;
            res = mid;
        } else {
            left = mid + 1;
        }
    }

    return res;
};
```

# Complexity
**Time:** O(n log S) - where S is the sum of array, binary search with O(n) check each iteration
**Space:** O(1) - only using variables
