# Sort Transformed Array
[360. Sort Transformed Array](https://leetcode.com/problems/0360.sort-transformed-array/)

Given a sorted integer array nums and three integers a, b and c, apply a quadratic function of the form f(x) = ax2 + bx + c to each element nums[i] in the array, and return the array in a sorted order.

Example 1:

Input: nums = [-4,-2,2,4], a = 1, b = 3, c = 5
Output: [3,9,15,33]
Example 2:

Input: nums = [-4,-2,2,4], a = -1, b = 3, c = 5
Output: [-23,-5,1,7]

# Key insights
* Two pointers approach from both ends to avoid sorting after transformation
* Parabola direction determines fill order: aâ‰¥0 fills backwards, a<0 fills forwards
* Direct result array placement eliminates need for additional sorting step
* O(n) time complexity by leveraging quadratic function properties

# Code
```javascript
var sortTransformedArray = function(nums, a, b, c) {
    let l = 0, r = nums.length - 1;
    const res = [];
    let idx = a >= 0 ? nums.length - 1 : 0;
    while (l <= r) {
        const val1 = quadratic(nums[l], a, b, c), val2 = quadratic(nums[r], a, b, c);
        if (a >= 0) {
            if (val1 >= val2) {
                res[idx--] = val1;
                ++l;
            } else {
                res[idx--] = val2;
                --r;
            }
        } else {
            if (val1 <= val2) {
                res[idx++] = val1;
                ++l;
            } else {
                res[idx++] = val2;
                --r;
            }
        }
    }
    return res;
};
function quadratic(x, a, b, c) {
    return a * x * x + b * x + c;
}
```

# Complexity
**Time:** O(n) - single pass through the array using two pointers, with constant time quadratic function evaluation at each step
**Space:** O(n) - creates result array of same size as input array
