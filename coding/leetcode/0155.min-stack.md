# Min Stack
[155. Min Stack](https://leetcode.com/problems/min-stack/)

Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

# Key insights
* Use two stacks: one for values and one for tracking the current minimum
* On push, also push to the min stack if the value is <= the current minimum
* On pop, also pop from the min stack if the popped value equals the current minimum
* getMin simply returns the top of the min stack

# Code
```javascript
var MinStack = function() {
    this.stack = [];
    this.minStack = [];
};

/** 
 * @param {number} val
 * @return {void}
 */
MinStack.prototype.push = function(val) {
    this.stack.push(val);
    if (this.minStack.length === 0) {
        this.minStack.push(val);
    } else {
        const min = this.minStack.at(-1);
        this.minStack.push(Math.min(min, val));
    }
};

/**
 * @return {void}
 */
MinStack.prototype.pop = function() {
    this.stack.pop();
    this.minStack.pop();
};

/**
 * @return {number}
 */
MinStack.prototype.top = function() {
    return this.stack.at(-1);
};

/**
 * @return {number}
 */
MinStack.prototype.getMin = function() {
    return this.minStack.at(-1);
};
```

# Complexity
**Time:** O(1) - All operations are constant time.**Space:** O(n) - Both stacks can hold up to n elements.
