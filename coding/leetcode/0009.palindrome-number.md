# Palindrome Number
[9. Palindrome Number](https://leetcode.com/problems/palindrome-number/)

Given an integer x, return true if x is a palindrome, and false otherwise.

# Key insights
* Negative numbers are never palindromes
* Reverse the entire number by extracting digits with `% 10` and compare with original
* No string conversion needed â€” pure math approach

# Code
```javascript
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    if (x < 0) return false;
    if (x === 0) return true;
    // if (x % 10 === 0) return false;

    let reverted = 0;
    let old = x;
    while (x > 0) {
        reverted = reverted * 10 + (x % 10);
        x = Math.floor(x / 10);
    }
    return old === reverted;
};
```

# Complexity
**Time:** O(log x) - Number of digits in x.
**Space:** O(1)
