# 4Sum
[18. 4Sum](https://leetcode.com/problems/4sum/)

Given an array nums of n integers, return an array of all the unique quadruplets [nums[a], nums[b], nums[c], nums[d]] such that:

# Key insights
* Extend 3Sum pattern: sort, fix two elements with nested loops, two pointers for remaining pair
* Skip duplicates at each level (i, j, left, right) to avoid duplicate quadruplets
* Compute remainder = target - nums[i] - nums[j] to reduce the inner problem to 2Sum

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number[][]}
 */
var fourSum = function(nums, target) {
    const len = nums.length;
    const res = [];

    nums.sort((a, b) => a - b);

    for (let i = 0; i < len; i++) {
        if (i > 0 && nums[i] === nums[i - 1]) continue; // dedup
        for (let j = i + 1; j < len; j++) {
            if (j > i + 1 && nums[j] === nums[j - 1]) continue;
            
            let left = j + 1, right = len - 1;
            const remainder = target - nums[i] - nums[j];

            while (left < right) {
                if (nums[left] + nums[right] < remainder) {
                    left++;
                } else if (nums[left] + nums[right] > remainder) {
                    right--;
                } else {
                    res.push([nums[i], nums[j], nums[left], nums[right]]);
                    
                    //dedup
                    while (left < right && nums[left] === nums[left + 1]) left++;
                    while (left < right && nums[right] === nums[right - 1]) right--;

                    left++;
                    right--;
                }
            }
        }
    }
    return res;
};
```

# Complexity
**Time:** O(n^3) - Two nested loops O(n^2) plus two-pointer scan O(n) each.
**Space:** O(log n) - For the sorting algorithm's stack space.
