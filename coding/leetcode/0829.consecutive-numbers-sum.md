# Consecutive Numbers Sum
[829. Consecutive Numbers Sum](https://leetcode.com/problems/consecutive-numbers-sum/)

Given an integer n, return the number of ways you can write n as the sum of consecutive positive integers.

# Key insights
* A sum of k consecutive numbers starting from x: x + (x+1) + ... + (x+k-1) = k*x + k*(k-1)/2
* So n = k*x + k*(k-1)/2, which means x = (n - k*(k-1)/2) / k
* x must be a positive integer, so (n - k*(k-1)/2) must be positive and divisible by k
* Iterate k from 1 upward while k*(k-1)/2 < n

# Code
```javascript
var consecutiveNumbersSum = function(n) {
    if (n <= 2) return 1;

    const nums = [];
    for (let i = 0; i <= Math.floor(n / 2); i++) {
        nums.push(i + 1);
    }
    
    let left = 0;
    let sum = 0;
    let count = 1;

    for (let right = 0; right < nums.length; right++) {
        sum += nums[right];

        while (left <= right && sum > n) {
            sum -= nums[left];
            left++;
        }

        if (sum === n) {
            count++;
        }
    }

    return count;
}
```
```javascript
var consecutiveNumbersSum = function(n) {
    let count = 0;

    for (let k = 1; k * (k - 1) / 2 < n; k++) {
        const remainder = n - k * (k - 1) / 2;
        if (remainder % k === 0) count++;
    }

    return count;
};
```

# Complexity
**Time:** O(sqrt(n)) - The loop runs while k*(k-1)/2 < n, so k is at most O(sqrt(n)).**Space:** O(1) - Only constant extra space is used.
