# Rotate String
[796. Rotate String](https://leetcode.com/problems/rotate-string/)

Simple Check:

# Key insights
* Concatenate string A with itself to create all possible rotations in one structure
* Use string search to check if B exists as substring in doubled string AA
* Length check ensures valid rotation before expensive substring search
* Transforms rotation problem into substring search with O(n) time complexity

# Code
```javascript
var rotateString = function(A, B) {
    return A.length === B.length && `${A}${A}`.indexOf(B) !== -1;
};
```
