# Spiral Matrix II
[59. Spiral Matrix II](https://leetcode.com/problems/spiral-matrix-ii/)

Given a positive integer n, generate an n x n matrix filled with elements from 1 to n2 in spiral order.

# Key insights
- Use four boundary pointers (top, bottom, left, right) to define the current spiral layer
- Fill numbers 1 to n^2 by traversing right, down, left, up in order, shrinking boundaries after each direction
- The `num <= n*n` guard in each loop handles odd-sized centers naturally

# Code
```javascript
/**
 * @param {number} n
 * @return {number[][]}
 */
var generateMatrix = function(n) {
    let top = 0;
    let bottom = n - 1;
    let left = 0;
    let right = n - 1;
    const board = Array(n).fill().map(() => Array(n).fill(0));
    let num = 1;

    while (num <= n*n) {
        for (let i = left; i <= right && num <= n*n; i++) {
            board[top][i] = num++;
        }
        top++;
        for (let i = top; i <= bottom && num <= n*n; i++) {
            board[i][right] = num++;
        }
        right--;
        for (let i = right; i >= left && num <= n*n; i--) {
            board[bottom][i] = num++;
        }
        bottom--;
        for (let i = bottom; i >= top && num <= n*n; i--) {
            board[i][left] = num++;
        }
        left++;
    }
    return board;
};

```

# Complexity
**Time:** O(n^2)**Space:** O(n^2) for the output matrix
