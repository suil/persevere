# Average of Levels in Binary Tree
[637. Average of Levels in Binary Tree](https://leetcode.com/problems/average-of-levels-in-binary-tree/)

Given the root of a binary tree, return the average value of the nodes on each level in the form of an array. Answers within 10-5 of the actual answer will be accepted.
 
Example 1:

# Key insights
- BFS level-order traversal: process all nodes at the same level together to compute the sum and average
- Use reduce to sum node values at each level, then divide by the level's node count
- Build the next level by collecting left and right children of all current-level nodes

# Code
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var averageOfLevels = function(root) {
    let queue = [];
    const res = [];

    if (root) queue.push(root);
    while (queue.length > 0) {
        const sum = queue.reduce((sum, n) => sum += n.val, 0); // get average
        console.log({sum})
        res.push(sum / queue.length);
        
        const nextQueue = [];
        for (const node of queue) {
            node.left && nextQueue.push(node.left);
            node.right && nextQueue.push(node.right);
        }
        queue = nextQueue;
    }

    return res;
};
```

# Complexity
**Time:** O(n) -- every node is visited once
**Space:** O(w) where w is the maximum width of the tree
