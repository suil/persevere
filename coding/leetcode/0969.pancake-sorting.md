# Pancake Sorting
[969. Pancake Sorting](https://leetcode.com/problems/pancake-sorting/)

Given an array of integers arr, sort the array by performing a series of pancake flips.

# Key insights
- Place elements from largest to smallest into their correct positions
- For each target value: find it, flip to bring it to front, flip again to move to correct position
- Each element requires at most 2 flips to place correctly
- Skip if element is already in correct position

# Code
```javascript
/**
 * @param {number[]} arr
 * @return {number[]}
 */
var pancakeSort = function(arr) {
    function flip(arr, k) {
        let left = 0;
        let right = k - 1;

        while (left < right) {
            [arr[left], arr[right]] = [arr[right], arr[left]];
            left++;
            right--;
        }
    }

    const res = [];
    for (let target = arr.length; target >= 1; target--) {
        let maxIndex = 0;
        for (let i = 0; i < arr.length; i++) {
            if (arr[i] === target) {
                maxIndex = i;
                break;
            }
        }

        if (maxIndex === target - 1) continue;

        if (maxIndex !== 0) {
            flip(arr, maxIndex + 1);
            res.push(maxIndex + 1);
        }

        flip(arr, target);
        res.push(target);
    }

    return res;
};
```

# Complexity
**Time:** O(n^2) - for each of n elements, finding and flipping takes O(n)
**Space:** O(n) - for storing the flip sequence
