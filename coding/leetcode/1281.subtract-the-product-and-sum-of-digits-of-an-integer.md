# Subtract the Product and Sum of Digits of an Integer
[1281. Subtract the Product and Sum of Digits of an Integer](https://leetcode.com/problems/subtract-the-product-and-sum-of-digits-of-an-integer/)

Given an integer number n, return the difference between the product of its digits and the sum of its digits.
 
Example 1:

# Key insights
- Extracts digits one by one using modulo 10 and integer division
- Accumulates the product and sum of digits in a single pass
- Returns product minus sum after all digits are processed

# Code
```javascript
/**
 * @param {number} n
 * @return {number}
 */
var subtractProductAndSum = function(n) {
    let product = 1;
    let sum = 0;
    
    while (n > 0) {
        const digit = n % 10;
        product *= digit;
        sum += digit;
        n = Math.floor(n / 10);
    }
    
    return product - sum;
};
```

# Complexity
**Time:** O(d) -- where d is the number of digits in n
**Space:** O(1) -- only a few variables
