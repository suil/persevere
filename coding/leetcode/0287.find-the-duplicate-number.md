# Find the Duplicate Number
[287. Find the Duplicate Number](https://leetcode.com/problems/find-the-duplicate-number/)

Given an array of integers nums containing n + 1 integers where each integer is in the range [1, n] inclusive.
There is only one repeated number in nums, return this repeated number.

Example 1:
Input: nums = [1,3,4,2,2]
Output: 2

Example 2:
Input: nums = [3,1,3,4,2]
Output: 3

Example 3:
Input: nums = [1,1]
Output: 1

Example 4:
Input: nums = [1,1,2]
Output: 1

# Key insights
* Binary search on value range (1 to n) instead of array indices
* Count elements â‰¤ mid to determine which half contains the duplicate
* Pigeonhole principle: if count > mid, duplicate must be in [1, mid] range
* O(n log n) time with O(1) space - no extra data structures needed

# Code
```javascript
var findDuplicate = function(nums) {
    let l = 1, h = nums.length - 1;
    while (l <= h) {
        let mid = Math.floor(l + (h - l) / 2);
        let cnt = 0;
        for (let i = 0; i < nums.length; i++) {
            if (nums[i] <= mid) {
                cnt++;
                console.log({i, 'nums[i]': nums[i], mid, cnt})
            }
        }
        if (cnt > mid) {
            h = mid - 1;
        } else {
            l = mid + 1;
        }
    }
    return l;
};
```
