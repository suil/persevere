# Longest Substring Without Repeating Characters
[3. Longest Substring Without Repeating Characters](https://leetcode.com/problems/longest-substring-without-repeating-characters/)

Given a string s, find the length of the longest substring without duplicate characters.

# Key insights
* Sliding window: expand `right` each step, shrink `left` when a duplicate is found
* Use a frequency map to track character counts in the current window
* When a duplicate exists, remove the leftmost character and advance `left` by one
* Only shrink by one per iteration (not a while loop), so the window size never decreases â€” it either grows or slides forward

# Code
```javascript
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLongestSubstring = function(s) {
    function hasDup(map) {
        for (const [char, count] of map) {
            if (count > 1) return true;
        }
        return false;
    }

    const map = new Map();
    
    let maxLen = 0;
    let left = 0;
    for (let right = 0; right < s.length; right++) {
        map.set(s[right], (map.get(s[right]) || 0) + 1);

        if (hasDup(map)) {
            map.set(s[left], map.get(s[left]) - 1);
            if (map.get(s[left]) === 0) map.delete(s[left]);
            left++;
        }

        maxLen = Math.max(maxLen, right - left + 1);
    }

    return maxLen
};
```

# Complexity
**Time:** O(n * m) - Where n is the string length and m is the map size (at most the character set size), since `hasDup` scans the map each iteration.
**Space:** O(m) - For the frequency map, where m is the number of distinct characters in the window.
