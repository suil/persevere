# 4Sum II
[454. 4Sum II](https://leetcode.com/problems/4sum-ii/)

Given four integer arrays nums1, nums2, nums3, and nums4 all of length n, return the number of tuples (i, j, k, l) such that:

# Key insights
- Split the 4 arrays into two groups of 2
- Store all possible sums of (nums1 + nums2) in a map with their counts
- For each sum of (nums3 + nums4), look for the complement (-sum) in the map
- Total count is sum of all complement matches

# Code
```javascript
/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @param {number[]} nums3
 * @param {number[]} nums4
 * @return {number}
 */
var fourSumCount = function(nums1, nums2, nums3, nums4) {
    const map = new Map();
    for (const num1 of nums1) {
        for (const num2 of nums2) {
            const sum = num1 + num2;
            map.set(sum, (map.get(sum) || 0) + 1);
        }
    }

    let count = 0;
    for (const num3 of nums3) {
        for (const num4 of nums4) {
            const sum = num3 + num4;
            const remainder = 0 - sum;
            count += (map.get(remainder) || 0);
        }
    }
    return count;
};
```

# Complexity
**Time:** O(n^2) - two nested loops for each pair of arrays
**Space:** O(n^2) - map stores up to n^2 different sums
