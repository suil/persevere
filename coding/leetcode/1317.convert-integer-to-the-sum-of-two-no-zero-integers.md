# Convert Integer to the Sum of Two No-Zero Integers
[1317. Convert Integer to the Sum of Two No-Zero Integers](https://leetcode.com/problems/convert-integer-to-the-sum-of-two-no-zero-integers/)

No-Zero integer is a positive integer that does not contain any 0 in its decimal representation.

# Key insights
- Brute force: iterates a from 1 to n-1, checking if both a and n-a contain no zero digits
- Uses string conversion and includes('0') to check for zero digits
- Returns the first valid pair found

# Code
```javascript
/**
 * @param {number} n
 * @return {number[]}
 */
var getNoZeroIntegers = function(n) {
    
    function hasZero(a) {
        return a.toString().includes('0');
    }

    for (let a = 1; a < n; a++) {
        let b = n - a;
        if (!hasZero(a) && !hasZero(b)) {
            return [a, b];
        }
    }
    
    return [];
};
```

# Complexity
**Time:** O(n * d) -- iterating up to n values, each checked in O(d) where d is the number of digits
**Space:** O(d) -- for string conversion
