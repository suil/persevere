# Squares of a Sorted Array
[977. Squares of a Sorted Array](https://leetcode.com/problems/squares-of-a-sorted-array/)

Given an integer array nums sorted in non-decreasing order, return an array of the squares of each number sorted in non-decreasing order.

# Key insights
- Two pointers at both ends of the sorted array; the largest squared value must be at one of the extremes
- Compares absolute values from left and right pointers, placing the larger square at the front via unshift
- Avoids the need for a separate sort after squaring

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var sortedSquares = function(nums) {
    const res = [];
    let left = 0;
    let right = nums.length - 1;
    while (left <= right) {
        if (nums[left] ** 2 > nums[right] ** 2) {
            res.unshift(nums[left] ** 2);
            left++;
        } else {
            res.unshift(nums[right] ** 2);
            right--;
        }
    }
    return res;
};
```

# Complexity
**Time:** O(n) -- each element is visited once by the two pointers
**Space:** O(n) -- the result array
