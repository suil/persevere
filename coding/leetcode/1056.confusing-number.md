# Confusing Number
[1056. Confusing Number](https://leetcode.com/problems/confusing-number/)

A confusing number is a number that when rotated 180 degrees becomes a different number with each digit valid.

# Key insights
- Maps each digit to its 180-degree rotation (0->0, 1->1, 6->9, 8->8, 9->6); digits 2,3,4,5,7 are invalid
- Builds the rotated number by reversing the digit order and applying the rotation map
- Returns true only if all digits are valid AND the rotated result differs from the original

# Code
```javascript
/**
 * @param {number} N
 * @return {boolean}
 */
var confusingNumber = function(N) {
    const map = new Map([
        ['0', '0'],
        ['1', '1'],
        ['9', '6'],
        ['8', '8'],
        ['6', '9'],
    ]);
    
    N = `${N}`;
    const reverted = [];
    
    for (let i = 0; i < N.length; i++) {
        if (!map.has(N[i])) {
            return false;
        }
        reverted.unshift(map.get(N[i]));
    }
    
    return reverted.join('') !== N;
};
```

# Complexity
**Time:** O(d) -- where d is the number of digits in N
**Space:** O(d) -- for the reverted array and string conversion
