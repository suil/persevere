# Valid Palindrome
[125. Valid Palindrome](https://leetcode.com/problems/valid-palindrome/)

# Key insights
- Uses two-pointer technique after preprocessing the string
toLowerCase() + regex replace(/[^a-z0-9]/g, '') sanitizes input in one pass
- Compares characters from both ends moving inward
- Early returns false on first mismatch

# Code
```javascript
var isPalindrome = function(s) {
    s = s.toLowerCase().replace(/[^a-z0-9]/g, '');
    let lo = 0, hi = s.length - 1;
    while (lo < hi) {
        if (s[lo] !== s[hi]) { return false; }
        lo++;
        hi--;
    }
    return true;
};
```

# Complexity
- **Time**: Time: O(n)
- **Space**: Space: O(n)
