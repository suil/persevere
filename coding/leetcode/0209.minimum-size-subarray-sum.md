# Minimum Size Subarray Sum
[209. Minimum Size Subarray Sum](https://leetcode.com/problems/minimum-size-subarray-sum/)

Given an array of positive integers nums and a positive integer target, return the minimal length of a subarray whose sum is greater than or equal to target. If there is no such subarray, return 0 instead.

# Key insights
- Sliding window: expand right pointer to grow the sum, shrink left pointer when sum >= target
- Track the minimum window length each time the sum meets or exceeds target before shrinking
- Each element is added and removed at most once, giving linear time

# Code
```javascript
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
var minSubArrayLen = function(target, nums) {
    let left = 0;
    let sum = 0;
    let minLen = Infinity;

    for (let i = 0; i < nums.length; i++) {
        sum += nums[i];
        while (sum >= target) {
            minLen = Math.min(minLen, i - left + 1);
            sum -= nums[left];
            left++;
        }
    }
    return minLen === Infinity ? 0 : minLen;
};
```

# Complexity
**Time:** O(n)
**Space:** O(1)
