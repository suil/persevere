# Maximum Average Subarray I
[643. Maximum Average Subarray I](https://leetcode.com/problems/maximum-average-subarray-i/)

You are given an integer array nums consisting of n elements, and an integer k.

# Key insights
- Fixed-size sliding window: compute the initial window sum, then slide by adding the new element and subtracting the outgoing element
- Track the maximum sum rather than average during iteration; divide by k only at the end

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var findMaxAverage = function(nums, k) {
    let sum = 0;
    for (let i = 0; i < k; i++) {
        sum += nums[i];
    }

    let maxSum = sum
    for (let i = k; i < nums.length; i++) {
        sum += nums[i] - nums[i - k];
        maxSum = Math.max(maxSum, sum);
    }

    return maxSum / k
};
```

# Complexity
**Time:** O(n) -- single pass through the array
**Space:** O(1)
