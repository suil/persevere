# Backspace String Compare
[844. Backspace String Compare](https://leetcode.com/problems/backspace-string-compare/)

Given two strings s and t, return true if they are equal when both are typed into empty text editors. '#' means a backspace character.

# Key insights
- Use a stack to simulate typing: push characters, pop on '#' (backspace)
- Process both strings independently, then compare the resulting stack contents
- Join stack to string for simple equality comparison

# Code
```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var backspaceCompare = function(s, t) {
    
    function process(str) {
        const stack = [];
        for (let i = 0; i < str.length; i++) {
            if (str[i] === '#') {
                stack.pop();
            } else {
                stack.push(str[i])
            }
        }
        return stack.join('');
    }

    const newS = process(s);
    const newT = process(t);
    return newS === newT;
};
```

# Complexity
**Time:** O(n + m) where n and m are the lengths of s and t
**Space:** O(n + m) for the two stacks
