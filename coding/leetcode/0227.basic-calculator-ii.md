# Basic Calculator II
[227. Basic Calculator II](https://leetcode.com/problems/basic-calculator-ii/)

Given a string s which represents an expression, evaluate this expression and return its value. 

# Key insights
- Track prevOp and apply it when the next operator (or end of string) is encountered
- For + and -, push the number onto the stack and add/subtract from running result
- For * and /, pop the last number, undo its effect on the result, compute the new value, then push and add it back

# Code
```javascript
/**
 * @param {string} s
 * @return {number}
 */
var calculate = function(s) {
    s = s.trim();
    let num = 0;
    let prevOp = '+';
    let res = 0;
    const stack = [];
    for (let i = 0; i < s.length; i++) {
        const char = s[i];
        if (char === ' ') continue;

        if (/\d/.test(char)) {
            num = num * 10 + Number(char);
        }

        if (i === s.length - 1 || ['+', '-', '*', '/'].includes(char)) {
            if (prevOp === '+') {
                res += num;
                stack.push(num);
            } else if (prevOp === '-') {
                res -= num;
                stack.push(-num);
            } else if (prevOp === '*') {
                const lastNum = stack.pop();
                res -= lastNum;
                const newNum = lastNum * num;
                res += newNum;
                stack.push(newNum);
            } else if (prevOp === '/') {
                const lastNum = stack.pop();
                res -= lastNum;
                const newNum = Math.sign(lastNum) * Math.floor(Math.abs(lastNum) / num);
                res += newNum;
                stack.push(newNum);
            }
            prevOp = char;
            num = 0;
        }
    }
    console.log(stack)
    return res;
};

```
