# Divide Array in Sets of K Consecutive Numbers
[1296. Divide Array in Sets of K Consecutive Numbers](https://leetcode.com/problems/divide-array-in-sets-of-k-consecutive-numbers/)

Given an array of integers nums and a positive integer k, check whether it is possible to divide this array into sets of k consecutive numbers.

# Key insights
- Sorts the array and uses a frequency map to greedily form groups of k consecutive numbers starting from the smallest available
- For each unprocessed number, tries to consume k consecutive values by decrementing their counts
- If any consecutive value is unavailable (count is 0), the division is impossible

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {boolean}
 */
var isPossibleDivide = function(nums, k) {
    nums.sort((a, b) => a - b);

    const freqMap = new Map();
    for (const num of nums) {
        freqMap.set(num, (freqMap.get(num) || 0) + 1);
    }

    for (const num of nums) {
        const count = freqMap.get(num);
        if (count <= 0) continue;

        for (let i = 0; i < k; i++) {
            const nextNum = num + i;
            if ((freqMap.get(nextNum) || 0) === 0) { return false; }
            freqMap.set(nextNum, (freqMap.get(nextNum) || 0) - 1);
        }
    }
    return true;
};
```

# Complexity
**Time:** O(n * k + n log n) -- sorting plus iterating with k-length group formation
**Space:** O(n) -- for the frequency map
