# N-Queens
[51. N-Queens](https://leetcode.com/problems/n-queens/)

The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other.

# Key insights
- Backtracking row by row, trying each column for the current row
- Use three Sets to track occupied columns, positive diagonals (row+col), and negative diagonals (row-col) for O(1) conflict checks
- When row === n, a valid placement is found; snapshot the board as strings

# Code
```javascript
/**
 * @param {number} n
 * @return {string[][]}
 */
var solveNQueens = function(n) {
    const board = Array(n).fill().map(() => Array(n).fill('.'));
    const colSet = new Set();
    const posDiagSet = new Set();
    const negDiagSet = new Set();

    const res = [];
    function backtrack(row) {
        if (row === n) {
            res.push(board.map(row => row.join('')));
            return;
        }

        for (let col = 0; col < n; col++) {
            if (colSet.has(col) || posDiagSet.has(row + col) || negDiagSet.has(row - col)) {
                continue;
            }

            colSet.add(col);
            posDiagSet.add(row + col);
            negDiagSet.add(row - col);
            board[row][col] = 'Q';
            backtrack(row + 1);
            colSet.delete(col);
            posDiagSet.delete(row + col);
            negDiagSet.delete(row - col);
            board[row][col] = '.';
        }
    }
    backtrack(0);
    return res;
};
```

# Complexity
**Time:** O(n!)**Space:** O(n^2) for the board plus O(n) for the Sets and recursion stack
