# Add Bold Tag in String
[616. Add Bold Tag in String](https://leetcode.com/problems/add-bold-tag-in-string/)

You are given a string s and an array of strings words.

# Key insights
- Use a boolean flag array to mark each character position that should be bold (covered by any word match)
- Find all occurrences of each word using indexOf in a loop, marking the corresponding range in the flag array
- Scan the flag array to merge adjacent/overlapping bold regions into single `<b>` tags

# Code
```javascript
/**
 * @param {string} s
 * @param {string[]} words
 * @return {string}
 */
var addBoldTag = function(s, words) {
    const flag = Array(s.length).fill(false);

    for (const word of words) {
        let index = s.indexOf(word);
        while (index !== -1) {
            for (let i = index; i < word.length + index; i++) flag[i] = true;
            index = s.indexOf(word, index + 1);
        }
    }

    let res = '';
    let i = 0;
    while (i < flag.length) {
        if (flag[i] === true) {
            res += '<b>';
            while (i < flag.length && flag[i] === true) {
                res += s[i];
                i++;
            }
            res += '</b>';
        } else {
            res += s[i];
            i++;
        }
    }
    return res;
};
```

# Complexity
**Time:** O(s * w * l) where s = |string|, w = number of words, l = average word length (indexOf scanning)
**Space:** O(s) -- for the boolean flag array and result string
