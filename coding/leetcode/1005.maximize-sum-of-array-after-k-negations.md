# Maximize Sum Of Array After K Negations
[1005. Maximize Sum Of Array After K Negations](https://leetcode.com/problems/maximize-sum-of-array-after-k-negations/)

Given an integer array nums and an integer k, modify the array in the following way:

# Key insights
- Sort the array and greedily negate negative numbers from smallest (most negative) first to maximize the sum
- If negations remain after flipping all negatives, and the count is odd, negate the smallest absolute value element
- Even remaining negations cancel out and can be ignored

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var largestSumAfterKNegations = function(nums, k) {
    nums.sort((a, b) => a - b);
    let minIdx;
    let min = Infinity;
    for (let i = 0; i < nums.length; i++) {
        if (k > 0 && nums[i] < 0) {
            nums[i] = -nums[i];
            k--;
        }
    }

    if (k > 0 && k % 2 === 1) { // odd
        for (let i = 0; i < nums.length; i++) {
            if (nums[i] < min) {
                min = nums[i];
                minIdx = i;
            }
        }
        nums[minIdx] = -nums[minIdx];
    }
    return nums.reduce((s, n) => s + n, 0);
};
```

# Complexity
**Time:** O(n log n) -- dominated by the initial sort
**Space:** O(1) -- sorting is in-place, only a few extra variables
