# 3Sum
[15. 3Sum](https://leetcode.com/problems/3sum/)

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0.

The solution set must not contain duplicate triplets.

# Key insights
* Sort the array first to enable two-pointer technique and skip duplicates
* Fix the first element, then use two pointers on the remaining subarray
* Skip duplicate values for the first element and for the two pointers to avoid duplicate triplets

# Code
```javascript
var threeSum = function(nums) {
    nums.sort((a, b) => a - b);
    const target = 0;
    const res = [];

    for (let i = 0; i < nums.length; i++) {
        if (i - 1 >= 0 && nums[i] === nums[i - 1]) continue; 

        let left = i + 1;
        let right = nums.length - 1;
        const sum = nums[i] + nums[left] + nums[right];

        while (left < right) {
            if (sum > target) {
                right--;
            } else if (sum < target) {
                left++;
            } else {
                res.push([nums[i], nums[left], nums[right]]);
                while (left + 1 < nums.length && nums[left] === nums[left + 1]) left++;
                while (right - 1 >= 0 && nums[right] === nums[right - 1]) right--;
                left++;
                right--;
            }
        }
    }

    return res;
};
```

# Complexity
**Time:** O(n^2) - Sorting takes O(n log n), then for each element the two-pointer scan is O(n).**Space:** O(1) - Aside from the output, only constant extra space is used (sorting is in-place).
