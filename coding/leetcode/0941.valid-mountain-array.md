# Valid Mountain Array
[941. Valid Mountain Array](https://leetcode.com/problems/valid-mountain-array/)

Given an array of integers arr, return true if and only if it is a valid mountain array.

# Key insights
- Climb up from the left while elements are increasing
- Climb up from the right while elements are increasing
- Valid mountain if both pointers meet at the same peak position
- Peak must not be at either end (need both ascending and descending parts)

# Code
```javascript
/**
 * @param {number[]} arr
 * @return {boolean}
 */
var validMountainArray = function(arr) {
    let left = 0;
    let right = arr.length - 1;

    while (left < arr.length && arr[left] < arr[left + 1]) left++;
    while (right > 0 && arr[right] < arr[right - 1]) right--;

    return left === right && left !== 0 && right != arr.length - 1;
};
```

# Complexity
**Time:** O(n) - two pointers traverse the array once
**Space:** O(1) - only using two pointers
