# Ransom Note
[383. Ransom Note](https://leetcode.com/problems/ransom-note/)

Given two strings ransomNote and magazine, return true if ransomNote can be constructed by using the letters from magazine and false otherwise.

# Key insights
- Build a frequency map of all letters in the magazine
- For each letter in ransom note, check if it's available in the map
- Decrement count as letters are used; return false if any letter is unavailable

# Code
```javascript
/**
 * @param {string} ransomNote
 * @param {string} magazine
 * @return {boolean}
 */
var canConstruct = function(ransomNote, magazine) {
    const map = new Map();
    for (const letter of magazine) {
        map.set(letter, (map.get(letter) || 0) + 1);
    }

    for (const letter of ransomNote) {
        if (!map.has(letter) || map.get(letter) === 0) { return false; }
        map.set(letter, map.get(letter) - 1);
    }
    return true;
};
```

# Complexity
**Time:** O(m + n) - where m is magazine length, n is ransom note length
**Space:** O(1) - map contains at most 26 lowercase letters
