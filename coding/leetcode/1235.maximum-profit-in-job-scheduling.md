# Maximum Profit in Job Scheduling
[1235. Maximum Profit in Job Scheduling](https://leetcode.com/problems/maximum-profit-in-job-scheduling/)


We have n jobs, where every job is scheduled to be done from startTime[i] to endTime[i], obtaining a profit of profit[i].

You're given the startTime, endTime and profit arrays, return the maximum profit you can take such that there are no two jobs in the subset with overlapping time range.

If you choose a job that ends at time X you will be able to start another job that starts at time X.

## Example 1:
![example 1 image](https://assets.leetcode.com/uploads/2019/10/10/sample1_1584.png)

Input: startTime = [1,2,3,3], endTime = [3,4,5,6], profit = [50,10,40,70]
Output: 120
Explanation: The subset chosen is the first and fourth job. 
Time range [1-3]+[3-6] , we get profit of 120 = 50 + 70.

## Example 2:
![example 2 image](https://assets.leetcode.com/uploads/2019/10/10/sample22_1584.png)

Input: startTime = [1,2,3,4,6], endTime = [3,5,10,6,9], profit = [20,20,100,70,60]
Output: 150
Explanation: The subset chosen is the first, fourth and fifth job. 
Profit obtained 150 = 20 + 70 + 60.

# Key insights
* Combine start time, end time and profit into job objects
* Sort job objects by start time
* back tracking on the array of job objects
  - Take it: Add its profit + max profit from jobs that end before it starts
  - Skip it: Keep the max profit from previous jobs
* when searching for next job, use the binary search to determine the first 
job in the array that is just later than the current end time.

    function findNext(index, time) {
        let left = index, right = len;
        while (left < right) {
            let mid = Math.floor((left + right) / 2);
            if (jobs[mid].start >= time) right = mid;
            else left = mid + 1;
        }
        return left;
    }

    function backtrack(index, memo = {}) {
        if (index >= len) return 0;

        if (index in memo) return memo[index];

        // Option 1: Don't include the current job
        let res = backtrack(index + 1, memo);

        // Option 2: Include the current job
        // let nextIndex = index + 1;
        // while (nextIndex < len && jobs[index].end > jobs[nextIndex].start) nextIndex++;

        res = Math.max(res, jobs[index].profit + backtrack(nextIndex, memo));
        memo[index] = res;
        return res;
    }
    return backtrack(0);
