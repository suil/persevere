# Matchsticks to Square
[473. Matchsticks to Square](https://leetcode.com/problems/matchsticks-to-square/)

You are given an integer array matchsticks where matchsticks[i] is the length of the ith matchstick. You want to use all the matchsticks to make one square. You should not break any stick, but you can link them up, and each matchstick must be used exactly one time.

# Key insights
- Early pruning: if total sum is not divisible by 4 or fewer than 4 sticks, return false immediately
- Use backtracking to assign each matchstick to one of 4 sides; when a side reaches the target length, start filling the next side
- Only need to successfully form 3 sides; the 4th is guaranteed by the remaining sum

# Code
```javascript
/**
 * @param {number[]} matchsticks
 * @return {boolean}
 */
var makesquare = function(matchsticks) {
    const n = matchsticks.length;
    if (n < 4) {
        return false;
    }

    const sum = matchsticks.reduce((acc, num) => acc + num, 0);
    if (sum % 4 !== 0) {
        return false;
    }

    const side = sum / 4;
    const visited = new Array(n).fill(false);

    function backtrack(index, count, currentSum) {
        if (count === 3) {
            return true; // If we formed 3 sides, the 4th will automatically be the remaining sum
        }

        if (currentSum === side) {
            return backtrack(0, count + 1, 0);
        }

        if (currentSum > side) {
            return false;
        }

        for (let i = index; i < n; i++) {
            if (visited[i] === true) continue;
            visited[i] = true;
            if (backtrack(i + 1, count, currentSum + matchsticks[i])) {
                return true;
            }
            visited[i] = false; // Backtrack
            // // Optimization to avoid duplicate combinations
            // while (i + 1 < n && matchsticks[i] === matchsticks[i + 1]) {
            //     i++;
            // }
        }

        return false;
    }

    return backtrack(0, 0, 0);
};

```

# Complexity
**Time:** O(4^n) -- each matchstick can be placed on one of 4 sides in the worst case
**Space:** O(n) -- recursion depth plus the visited array
