# Reverse Vowels of a String
[345. Reverse Vowels of a String](https://leetcode.com/problems/reverse-vowels-of-a-string/)

Given a string s, reverse only all the vowels in the string and return it.

# Key insights
- Two pointers from both ends of the string
- Move each pointer inward until it finds a vowel
- When both pointers are on vowels, swap them
- Use a Set for O(1) vowel lookup (include both upper and lowercase)

# Code
```javascript
/**
 * @param {string} s
 * @return {string}
 */
var reverseVowels = function(s) {
    s = s.split('');
    const vowels = new Set(['a', 'e', 'i', 'o', 'u', 'A', 'E', 'I', 'O', 'U']);
    let left = 0, right = s.length - 1;

    while (left < right) {
        const leftChar = s[left];
        const rightChar = s[right];

        if (!vowels.has(leftChar)) {
            left++;
        } else if (!vowels.has(rightChar)) {
            right--;
        } else {
            [s[left], s[right]] = [s[right], s[left]];
            right--;
            left++;
        }
    }
    return s.join('');
};
```

# Complexity
**Time:** O(n) - each character is visited at most once
**Space:** O(n) - for converting string to array (strings are immutable in JS)
