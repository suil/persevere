# Symmetric Tree
[101. Symmetric Tree](https://leetcode.com/problems/symmetric-tree/)

Given the root of a binary tree, check whether it is a mirror of itself (i.e., symmetric around its center).

Example 1:
        1
      /   \
     2     2
    / \   / \
   3   4 3   4 
Input: root = [1,2,2,3,4,4,3]
Output: true

Example 2:
Input: root = [1,2,2,null,3,null,3]
Output: false

# Key insights
* Use recursive helper function to compare two subtrees simultaneously
* Check symmetry by comparing left subtree of one node with right subtree of mirror
* Handle null cases first to avoid runtime errors in recursive calls
* Mirror comparison: node1.left vs node2.right and node1.right vs node2.left

# Code
```javascript
var isSymmetric = function(root) {
    if (root === null) { return true; }
    return isMirror(root, root);
}
function isMirror(node1, node2) {
    if (node1 === null && node2 === null) { return true; }
    if (node1 === null || node2 === null) { return false; }
    return node1.val === node2.val
        && isMirror(node1.left, node2.right)
        && isMirror(node1.right, node2.left);
}
```
