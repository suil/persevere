# Key insights
- Three steps: find the middle with slow/fast pointers, reverse the second half, then merge/interleave both halves
- The reverse is done recursively, returning the new head of the reversed list
- Merge alternates nodes from the first half and reversed second half by rewiring next pointers

# Code
```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {void} Do not return anything, modify head in-place instead.
 */
var reorderList = function(head) {

    let slow = head;
    let fast = head;
    let pre = null

    while (fast && fast.next) {
        pre = slow;
        slow = slow.next;
        fast = fast.next.next;
    }

    let reversed = reverseList(slow.next);
    slow.next = null;
    
    while (head && reversed) {
        const headNext = head.next;
        const reversedNext = reversed.next;
        head.next = reversed;
        reversed.next = headNext;
        head = headNext;
        reversed = reversedNext;
    }
};

function reverseList(head) {
    if (head === null) return null;
    if (head.next === null) return head;

    const newTail = head.next;
    const reversedHeadNext = reverseList(head.next);
    newTail.next = head;
    head.next = null;
    return reversedHeadNext;
}
```

# Complexity
**Time:** O(n)
**Space:** O(n) due to recursive reverse (O(1) with iterative reverse)
