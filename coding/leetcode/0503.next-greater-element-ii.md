# Next Greater Element II
[503. Next Greater Element II](https://leetcode.com/problems/next-greater-element-ii/)

Given a circular integer array nums (i.e., the next element of nums[nums.length - 1] is nums[0]), return the next greater number for every element in nums.

# Key insights
- Use a monotonic decreasing stack storing indices; when a larger element is found, pop and assign it as the next greater element
- Iterate through the array twice (using modulo) to handle the circular nature of the array
- Initialize result with -1 so elements with no next greater value default correctly

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var nextGreaterElements = function(nums) {
    const stack = [];
    const res = Array(nums.length).fill(-1);

    for (let i = 0; i < nums.length * 2; i++) {
        const num = nums[i % nums.length];

        while (stack.length > 0 && nums[stack.at(-1)] < num) {
            const index = stack.pop();
            res[index] = nums[i % nums.length];
        }

        stack.push(i % nums.length);
    }
    return res;
};
```

# Complexity
**Time:** O(n) -- each element is pushed and popped from the stack at most twice
**Space:** O(n) -- for the stack and result array
