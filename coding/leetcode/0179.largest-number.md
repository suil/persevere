# Largest Number
[179. Largest Number](https://leetcode.com/problems/largest-number/)

Given a list of non-negative integers nums, arrange them such that they form the largest number and return it.

# Key insights
- Custom sort comparator: compare concatenations (b+a) vs (a+b) to determine which ordering produces a larger number
- Edge case: if the largest number after sorting is '0', the result is '0' (all zeros)
- Convert numbers to strings first, then join after sorting

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {string}
 */
var largestNumber = function(nums) {
    const strNums = nums.map(String);

    strNums.sort((a, b) => (b + a).localeCompare((a + b)));

    if (strNums[0] === '0') return '0';

    return strNums.join('');
};
```

# Complexity
**Time:** O(n log n * k) where k is the average digit count per number
**Space:** O(n) for the string array
