# Find K Closest Elements
[658. Find K Closest Elements](https://leetcode.com/problems/find-k-closest-elements/)

Given a sorted integer array arr, two integers k and x, return the k closest integers to x in the array. The result should also be sorted in ascending order.

# Key insights
- Binary search for the optimal left boundary of the k-length window in the sorted array
- Compare distances from x to arr[mid] and arr[mid+k]: if the left side is farther, move right; otherwise record mid as a candidate and move left
- The result is a contiguous subarray of length k starting from the found position

# Code
```javascript
var findClosestElements = function(arr, k, x) {
    let left = 0, right = arr.length - k;
    let ans = 0;
    while (left <= right) {
        let mid = Math.floor((left + right) / 2);
        if (x - arr[mid] > arr[mid + k] - x) {
            left = mid + 1;
        } else {
            ans = mid; // store possible answer
            right = mid - 1;
        }
    }
    return arr.slice(ans, ans + k);
};
```

# Complexity
**Time:** O(log(n - k) + k) -- binary search plus slicing the result
**Space:** O(k) -- for the output array
