# Longest String Chain
[1048. Longest String Chain](https://leetcode.com/problems/longest-string-chain/)

You are given an array of words where each word consists of lowercase English letters.

# Key insights
- For each word, tries removing one character at each position to find predecessors in the word set
- Uses memoized DFS (top-down DP) to compute the longest chain ending at each word
- A Set enables O(1) lookup to check if a predecessor word exists in the input

# Code
```javascript
/**
 * @param {string[]} words
 * @return {number}
 */
var longestStrChain = function(words) {
    const set = new Set(words);

    function backtrack(word, memo = {}) {
        if (word in memo) return memo[word];

        let maxLen = 1;
        for (let i = 0; i < word.length; i++) {
            const nextWord = word.substring(0, i) + word.substring(i + 1);
            if (set.has(nextWord)) {
                maxLen = Math.max(maxLen, backtrack(nextWord, memo) + 1);
            }
        }
        memo[word] = maxLen;
        return maxLen;
    }

    let maxLen = 0;
    for (const word of words) {
        maxLen = Math.max(maxLen, backtrack(word))
    }
    return maxLen;
};
```

# Complexity
**Time:** O(n * L^2) -- for each of n words, generate up to L predecessors each taking O(L) to create via substring
**Space:** O(n * L) -- for the memo object and the word set
