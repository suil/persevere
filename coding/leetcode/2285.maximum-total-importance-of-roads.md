# Maximum Total Importance of Roads
[2285. Maximum Total Importance of Roads](https://leetcode.com/problems/maximum-total-importance-of-roads/)

You are given an integer n denoting the number of cities in a country. The cities are numbered from 0 to n - 1.

# Code
```javascript
/**
 * @param {number} n
 * @param {number[][]} roads
 * @return {number}
 */
var maximumImportance = function(n, roads) {
    const degrees = new Map();
    
    for (const [a, b] of roads) {
        degrees.set(a, (degrees.get(a) || 0) + 1)
        degrees.set(b, (degrees.get(b) || 0) + 1)
    }

    const nodes = [...degrees.entries()];
    nodes.sort((a, b) => b[1] - a[1]);

    const importance = new Map();

    let importanceVal = n;
    for (const [node] of nodes) {
        importance.set(node, importanceVal--)
    }

    let res = 0;
    for (const [a, b] of roads) {
        res += importance.get(a) + importance.get(b);
    }

    return res;
};
```
