# Find Median from Data Stream
[295. Find Median from Data Stream](https://leetcode.com/problems/find-median-from-data-stream/)

The median is the middle value in an ordered integer list. If the size of the list is even, there is no middle value, and the median is the mean of the two middle values.

Implement the MedianFinder class:
- `MedianFinder()` initializes the object.
- `void addNum(int num)` adds the integer num from the data stream.
- `double findMedian()` returns the median of all elements so far.

Example 1:

Input: ["MedianFinder", "addNum", "addNum", "findMedian", "addNum", "findMedian"]
       [[], [1], [2], [], [3], []]
Output: [null, null, null, 1.5, null, 2.0]

Constraints:

-10^5 <= num <= 10^5
At most 5 * 10^4 calls will be made to addNum and findMedian.

# Key insights

- Use **two heaps**: a max-heap `lo` (lower half) and a min-heap `hi` (upper half)
- Invariant: `lo.size() >= hi.size()` and `lo.max <= hi.min` — the median is always at the boundary
- **addNum** in two steps:
  1. Push to `lo`, then move `lo`'s max to `hi` (maintains the ordering invariant)
  2. If `hi` grows larger, move `hi`'s min back to `lo` (rebalances sizes)
- **findMedian**: if sizes are equal, average both tops; otherwise return `lo`'s top (lo always holds the extra element)
- JavaScript lacks a built-in heap, so a generic `Heap` class with a comparator is needed

# Code
```javascript
var MedianFinder = function() {
    this.minQ = new PriorityQueue((a, b) => a - b);
    this.maxQ = new PriorityQueue((a, b) => b - a);
};

MedianFinder.prototype.addNum = function(num) {
    this.minQ.enqueue(num);
    this.maxQ.enqueue(this.minQ.dequeue());

    if (this.minQ.size() < this.maxQ.size()) {
        this.minQ.enqueue(this.maxQ.dequeue());
    }
};

MedianFinder.prototype.findMedian = function() {
    const minQSize = this.minQ.size();
    const maxQSize = this.maxQ.size();

    if (minQSize === 0 && maxQSize === 0) return null;

    if (minQSize < maxQSize) {
        return this.maxQ.front();
    } else if (minQSize > maxQSize) {
        return this.minQ.front();
    } else {
        return (this.minQ.front() + this.maxQ.front()) / 2;
    }
};
```
# Complexity

- **addNum**: O(log n) — each call does at most 2 heap push/pop operations
- **findMedian**: O(1) — just peek at the tops of both heaps
- **Space**: O(n) — all elements are stored across the two heaps
