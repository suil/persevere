# Maximum Depth of Binary Tree
[104. Maximum Depth of Binary Tree](https://leetcode.com/problems/maximum-depth-of-binary-tree/)

Given the root of a binary tree, return its maximum depth.

A binary tree's maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Example 1:
    3
   / \
  9   20
     /  \
   15    7
Input: root = [3,9,20,null,null,15,7]
Output: 3
Example 2:

Input: root = [1,null,2]
Output: 2
Example 3:

Input: root = []
Output: 0
Example 4:

Input: root = [0]
Output: 1

# Key insights
* Uses recursive divide-and-conquer to explore both subtrees
* Base case handles null nodes by returning depth 0
* Combines subproblem results with Math.max() + 1 for current level
* Time O(n) space O(h) where h is tree height due to call stack

# Code
```javascript
var maxDepth = function(root) {
    if (!root) {
        return 0;
    }
    return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
};
```
