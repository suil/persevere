# Strings Differ by One Character
[1554. Strings Differ by One Character](https://leetcode.com/problems/strings-differ-by-one-character/)

Given a list of strings dict where all the strings are of the same length.

# Key insights
- Compare all pairs of strings
- Count character differences between each pair
- Return true if any pair differs by exactly one character
- Could be optimized using hashing with wildcards

# Code
```javascript
/**
 * @param {string[]} dict
 * @return {boolean}
 */
var differByOne = function(dict) {
    const len = dict.length;

    const firstStr = dict[0];
    for (let i = 0; i < len; i++) {
        if (firstStr.length !== dict[i].length) return false;
    }
    
    function getDiff(str1, str2) {
        const strLen = str1.length;
        let count = 0;
        for (let i = 0; i < strLen; i++) {
            if (str1[i] !== str2[i]) count++;
        }
        return count;
    }

    for (let i = 0; i < len; i++) {
        for (let j = i + 1; j < len; j++) {
            const str1 = dict[i];
            const str2 = dict[j];
            if (getDiff(str1, str2) === 1) return true;
        }
    }

    return false;
};
```

# Complexity
**Time:** O(n^2 * m) - comparing all pairs, each comparison takes O(m)
**Space:** O(1) - only using variables
