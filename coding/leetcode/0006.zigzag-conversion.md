# Zigzag Conversion
[6. Zigzag Conversion](https://leetcode.com/problems/zigzag-conversion/)

The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

# Key insights
* Simulate the zigzag by distributing characters into `numRows` string buckets
* Track current row and direction; reverse direction when hitting row 0 or numRows-1
* Concatenate all row strings at the end for the result

# Code
```javascript
/**
 * @param {string} s
 * @param {number} numRows
 * @return {string}
 */
var convert = function(s, numRows) {
    const rows = Array(Math.max(numRows, s.length)).fill('');
    let row = 0;
    let rowDelta = -1; // 1 or -1

    for (const char of s) {
        rows[row] += char;
        
        if (row === 0 || row === numRows - 1) {
            rowDelta = (rowDelta === 1 ? -1 : 1);
        }
        row += rowDelta;
    }
    return rows.filter(Boolean).join('');
};
```

# Complexity
**Time:** O(n) - Single pass through the string.
**Space:** O(n) - Storing all characters across the row strings.
