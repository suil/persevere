# Next Permutation
[31. Next Permutation](https://leetcode.com/problems/next-permutation/)

A permutation of an array of integers is an arrangement of its members into a sequence or linear order.

# Key insights
- Find the rightmost element that is smaller than its next element (the "dip" at index i), scanning from right to left
- Swap nums[i] with the smallest element to its right that is still larger than nums[i]
- Reverse the suffix after index i to get the lexicographically next permutation
- If no dip exists (fully descending), reversing the whole array gives the smallest permutation

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
var nextPermutation = function(nums) {
    let i = nums.length - 2;
    while (i >= 0 && nums[i] >= nums[i + 1]) i--;

    if (i >= 0) {
        let j = nums.length - 1;
        while (nums[j] <= nums[i]) j--;
        [nums[i], nums[j]] = [nums[j], nums[i]];
    }

    let left = i + 1; 
    let right = nums.length - 1;
    while (left < right) {
        [nums[left], nums[right]] = [nums[right], nums[left]];
        left++;
        right--;
    }
};
```

# Complexity
**Time:** O(n)
**Space:** O(1)
