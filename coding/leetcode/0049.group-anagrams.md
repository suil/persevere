# Group Anagrams
[49. Group Anagrams](https://leetcode.com/problems/group-anagrams/)

Given an array of strings strs, group the anagrams together. You can return the answer in any order.

# Key insights
* Anagrams share the same sorted character sequence
* Use the sorted string as a hash key to group words together
* A map from sorted key to list of original strings gives the grouping

# Code
```javascript
var groupAnagrams = function(strs) {
    const map = new Map();

    for (const str of strs) {
        const sortedStr = str.split('').sort().join('')
        if (map.has(sortedStr)) {
            map.get(sortedStr).push(str);
        } else {
            map.set(sortedStr, [str]);
        }
    }

    const res = []
    for (const [key, arr] of map) {
        res.push(arr);
    }
    return res;
};
```

# Complexity
**Time:** O(n * k log k) - Where n is the number of strings and k is the maximum string length, due to sorting each string.
**Space:** O(n * k) - Storing all strings in the map.
