# Find Peak Element
[162. Find Peak Element](https://leetcode.com/problems/find-peak-element/)

A peak element is an element that is strictly greater than its neighbors.

# Key insights
- Binary search: if nums[mid] < nums[mid+1], a peak must exist to the right, so move left = mid+1
- Otherwise a peak exists at mid or to the left, so move right = mid-1
- When left > right, left points to the peak element

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var findPeakElement = function(nums) {
    if (nums.length === 1) return 0;

    let left = 0, right = nums.length - 1;
    while (left <= right) {
        const mid = left + Math.floor((right - left) / 2);
        if (nums[mid] < nums[mid + 1]) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return left;
};
```

# Complexity
**Time:** O(log n)**Space:** O(1)
