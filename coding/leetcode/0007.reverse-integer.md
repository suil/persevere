# Reverse Integer
[7. Reverse Integer](https://leetcode.com/problems/reverse-integer/)

Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range [-231, 231 - 1], then return 0.

# Key insights
* Extract digits from the end using `x % 10`, build reversed number by multiplying accumulator by 10
* Handle negative numbers separately by converting to positive first
* Check for 32-bit overflow during construction

# Code
```javascript
/**
 * @param {number} x
 * @return {number}
 */
var reverse = function(x) {
    const isNeg = x < 0;
    if (isNeg) x = -x;

    let reverted = 0;
    while (x > 0) {
        reverted = reverted * 10 + x % 10;
        x = Math.floor(x / 10);
        if (reverted > (2**31)){
            return 0
        }
    }

    return isNeg ? -reverted : reverted;
};
```

# Complexity
**Time:** O(log x) - Number of digits in x.
**Space:** O(1)
