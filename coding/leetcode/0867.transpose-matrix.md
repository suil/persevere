# Transpose Matrix
[867. Transpose Matrix](https://leetcode.com/problems/transpose-matrix/)

Given a 2D integer array matrix, return the transpose of matrix.

# Key insights
- Transpose swaps rows and columns: result[col][row] = matrix[row][col]
- Create a new matrix with swapped dimensions (colLen x rowLen)
- Iterate through original matrix and place elements in transposed positions

# Code
```javascript
/**
 * @param {number[][]} matrix
 * @return {number[][]}
 */
var transpose = function(matrix) {
    const rowLen = matrix.length;
    const colLen = matrix[0].length;
    const res = Array(colLen).fill().map(() => Array(rowLen).fill(0));

    for (let row = 0; row < rowLen; row++) {
        for (let col = 0; col < colLen; col++) {
            res[col][row] = matrix[row][col];
        }
    }
    return res;
};
```

# Complexity
**Time:** O(m * n) - visiting each element once
**Space:** O(m * n) - for the result matrix
