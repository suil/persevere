# Flatten Binary Tree to Linked List
[114. Flatten Binary Tree to Linked List](https://leetcode.com/problems/flatten-binary-tree-to-linked-list/)

Given the root of a binary tree, flatten the tree into a "linked list":

# Key insights
- Post-order style recursion: flatten left and right subtrees first, then rewire pointers
- If a left subtree exists, attach the right subtree to the left tail, then move the left subtree to the right and set left to null
- Return the rightmost tail node (rightTail if it exists, otherwise leftTail) for the parent to link

# Code
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {void} Do not return anything, modify root in-place instead.
 */
var flatten = function(root) {
    function flattenTree(node) {
        if (!node) return node;
        if (!node.left && !node.right) return node;

        const leftTail = flattenTree(node.left);
        const rightTail = flattenTree(node.right);

        if (leftTail) {
            leftTail.right = node.right;
            node.right = node.left;
            node.left = null
        }
        
        return rightTail ? rightTail : leftTail;
    }

    return flattenTree(root);
};
```

# Complexity
**Time:** O(n)
**Space:** O(n) for recursion stack in the worst case (skewed tree)
