# Max Chunks To Make Sorted
[769. Max Chunks To Make Sorted](https://leetcode.com/problems/max-chunks-to-make-sorted/)

You are given an integer array arr of length n that represents a permutation of the integers in the range [0, n - 1].

# Key insights
- Since the array is a permutation of [0, n-1], a chunk [0..i] can be independently sorted iff max(arr[0..i]) == i
- Track the running maximum; whenever it equals the current index, all values 0..i are contained in arr[0..i]
- Each such point marks the end of a valid chunk

# Code
```javascript
/**
 * @param {number[]} arr
 * @return {number}
 */
var maxChunksToSorted = function(arr) {
    let ans = 0, max = 0;
    for (let i = 0; i < arr.length; ++i) {
        max = Math.max(max, arr[i]);
        if (max == i) ans++;
    }
    return ans;
};
```

# Complexity
**Time:** O(n)**Space:** O(1)
