# Jump Game VI
[1696. Jump Game VI](https://leetcode.com/problems/jump-game-vi/)

You are given a 0-indexed integer array nums and an integer k.

# Key insights
- DP with sliding window maximum using monotonic deque
- dp[i] = maximum score to reach index i
- From each position, can jump to any of next k positions
- Deque stores indices with dp values in decreasing order; front is always max

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var maxResult = function(nums, k) {
    const n = nums.length;
    const dp = new Array(n).fill(0);
    dp[0] = nums[0];

    // deque stores indices, with dp values in decreasing order
    const deque = [0];

    for (let i = 1; i < n; i++) {
        // front of deque is max dp in range
        dp[i] = nums[i] + dp[deque[0]];

        // maintain deque: remove smaller dp values from back
        while (deque.length && dp[deque.at(-1)] <= dp[i]) {
            deque.pop();
        }
        deque.push(i);

        // remove indices out of window
        if (deque[0] <= i - k) {
            deque.shift();
        }
    }

    return dp[n - 1];
};
```

# Complexity
**Time:** O(n) - each index is added and removed from deque at most once
**Space:** O(n) - for dp array and deque
