# Base 7
[504. Base 7](https://leetcode.com/problems/base-7/)

Given an integer num, return a string of its base 7 representation.

# Key insights
- Repeatedly divide by 7 and collect remainders to build the base-7 digits from least significant to most significant
- Handle the sign separately by converting to positive first and prepending '-' if negative
- Reverse the collected digits at the end since they are generated in reverse order

# Code
```javascript
/**
 * @param {number} num
 * @return {string}
 */
var convertToBase7 = function(num) {
    if (num === 0) {
        return '0'
    }
    let isNagtive = false;
    if (num < 0) {
        isNagtive = true;
        num = -num;
    }
    let base7Str = [];
    while (num > 0) {
        base7Str.push(`${num % 7}`);
        num = Math.floor(num / 7);
    }
    base7Str = base7Str.reverse();
    return `${isNagtive ? '-' : ''}${base7Str.join('')}`;
};
```

# Complexity
**Time:** O(log_7(n)) -- number of digits in base 7
**Space:** O(log_7(n)) -- for storing the digits
