# Find the Difference
[389. Find the Difference](https://leetcode.com/problems/find-the-difference/)

You are given two strings s and t.

# Key insights
- Use a frequency array of size 26 for lowercase letters
- Add 1 for each character in s, subtract 1 for each character in t
- The character with count -1 is the added character (t has one extra)

# Code
```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {character}
 */
var findTheDifference = function(s, t) {
    let tracker = new Array(26);
    tracker.fill(0);
    
    let aCode = 97;
    
    for (let i = 0; i<s.length; i++) {
        tracker[s.charCodeAt(i) - aCode] +=1
        tracker[t.charCodeAt(i) - aCode] -=1;
    }

    tracker[t.charCodeAt(t.length - 1) - aCode] -=1;
    
    return String.fromCharCode(tracker.indexOf(-1) + aCode);
};
```

# Complexity
**Time:** O(n) - single pass through both strings
**Space:** O(1) - fixed array of 26 characters
