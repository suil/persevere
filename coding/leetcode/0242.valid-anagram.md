# Valid Anagram
[242. Valid Anagram](https://leetcode.com/problems/valid-anagram/)

Given two strings s and t, return true if t is an anagram of s, and false otherwise.

An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

Example 1:

Input: s = "anagram", t = "nagaram"
Output: true
Example 2:

Input: s = "rat", t = "car"
Output: false

Constraints:

1 <= s.length, t.length <= 5 * 104
s and t consist of lowercase English letters.

# Key insights

- Two strings are anagrams if and only if they have the same character **frequency counts**
- Early exit: if lengths differ, they cannot be anagrams
- Use a hash map to count characters in `s`, then decrement for each character in `t`; if any count goes below 0 (or key missing), return false
- Since input is limited to lowercase English letters, the map has at most 26 entries (O(1) space)

# Code
```javascript
var isAnagram = function(s, t) {
    if (s.length !== t.length) return false;
    const map = new Map();

    for (let i = 0; i < s.length; i++) {
        map.set(s[i], (map.get(s[i]) || 0) + 1);
        map.set(t[i], (map.get(t[i]) || 0) - 1);
    }
    
    return map.values().every(m => m === 0);
};
```
# Complexity

- **Time**: O(n) where n is the length of the strings
