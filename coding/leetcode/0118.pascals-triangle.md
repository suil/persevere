# Pascal's Triangle
[118. Pascal's Triangle](https://leetcode.com/problems/pascals-triangle/)

Given an integer numRows, return the first numRows of Pascal's triangle.

# Key insights
- Each row starts as all 1s; inner elements are computed as the sum of two elements from the previous row
- Row i has i+1 elements; the inner loop fills indices 1 to i-1 using res[i-1][j-1] + res[i-1][j]
- Build rows iteratively, referencing the already-completed previous row

# Code
```javascript
/**
 * @param {number} numRows
 * @return {number[][]}
 */
var generate = function(numRows) {
    const res = [];
    for (let i = 0; i < numRows; i++) {
        const nums = Array(i + 1).fill(1);
        for (let j = 1; j < i; j++) {
            nums[j] = res[i - 1][j - 1] + res[i - 1][j];
        }
        res.push(nums);
    }
    return res;
};
```

# Complexity
**Time:** O(numRows^2)
**Space:** O(numRows^2) for the output
