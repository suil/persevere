# Strobogrammatic Number
[246. Strobogrammatic Number](https://leetcode.com/problems/strobogrammatic-number/)

Given a string num which represents an integer, return true if num is a strobogrammatic number.

# Key insights
- A strobogrammatic number looks the same when rotated 180 degrees
- Only digits 0, 1, 6, 8, 9 are valid (0, 1, 8 map to themselves; 6 and 9 map to each other)
- Use two pointers from both ends, check if digits at opposite positions form valid pairs

# Code
```javascript
/**
 * @param {string} num
 * @return {boolean}
 */
var isStrobogrammatic = function(num) {
    let upsideDowns = {
        '1' : '1',
        '6' : '9',
        '8' : '8',
        '9' : '6',
        '0' : '0'
    }
    
    for (let i = 0, j = num.length -1; i < num.length; i++, j--){
        if (upsideDowns[num[i]] != num[j]) {
            return false
        }
    }
    
    return true
};
```

# Complexity
**Time:** O(n) - single pass through the string with two pointers
**Space:** O(1) - only using a fixed-size mapping object
