# Meeting Rooms
[253. Meeting Rooms](https://leetcode.com/problems/meeting-rooms-ii)

Given an array of meeting time intervals intervals where intervals[i] = [starti, endi], return the minimum number of conference rooms required.

# Key insights
* Sort intervals by start time to process meetings chronologically
* Track end times of active meetings to determine room availability
* Greedy approach: reuse earliest-ending room when possible
* Array length represents minimum rooms needed at any point

# Code
```javascript
var minMeetingRooms = function(intervals) {
    intervals.sort((a, b) => a[0] - b[0]);

    const pq = new PriorityQueue((a, b) => a[1] - b[1]);

    let count = 1;
    pq.enqueue(intervals[0]);
    for (let i = 1; i < intervals.length; i++) {
        const [, earliestEnd] = pq.front();
        const [currStart, currEnd] = intervals[i];
        if (currStart >= earliestEnd) {
            pq.pop();
        } else {
            count++;
        }
        pq.enqueue(intervals[i]);
    }
    return count;
};
```

# Complexity
**Time:** O(n² log n) - O(n log n) for sorting plus O(n²) for the loop where each iteration finds min and indexOf in O(n) time
**Space:** O(n) - meetingEndTimes array can grow up to size n in worst case
