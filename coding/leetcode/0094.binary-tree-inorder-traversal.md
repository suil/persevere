# Binary Tree Inorder Traversal
[94. Binary Tree Inorder Traversal](https://leetcode.com/problems/binary-tree-inorder-traversal/)

Recursive

Non-recursive

# Key insights
* Uses recursive divide-and-conquer pattern to break tree into left subtree, root, right subtree
* Leverages JavaScript spread operator to concatenate arrays from recursive calls
* Base case returns empty array when node is null to handle leaf nodes gracefully
* Time O(n) space O(h) where h is tree height due to recursion stack and array creation

# Code
```javascript
var inorderTraversal = function(root) {
    return inorderTraverse(root);
};
function inorderTraverse(node) {
    if (node === null) {
        return [];
    }
    return [
        ...inorderTraverse(node.left),
        node.val,
        ...inorderTraverse(node.right)
    ];
}
```
```javascript
var inorderTraversal = function(root) {
    const res = [];
    const stack = [];
    let pointer = root;
    while (stack.length > 0 || pointer !== null) {
        if (pointer !== null) {
            stack.push(pointer);
            pointer = pointer.left;
        } else {
            pointer = stack.pop();
            res.push(pointer.val);
            pointer = pointer.right;
        }
    }
    return res;
};
```
