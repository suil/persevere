# Diagonal Traverse
[498. Diagonal Traverse](https://leetcode.com/problems/diagonal-traverse/)

# Key insights
- Elements on the same diagonal have the same sum of indices (row + col)
- Diagonals alternate direction: even-indexed diagonals go up-right, odd-indexed diagonals go down-left
- Group elements by their diagonal index, then reverse the order for even diagonals using `unshift()` vs `push()`
- Total number of diagonals in an m×n matrix is m + n - 1

# Code
```javascript
var findDiagonalOrder = function(mat) {
    const m = mat.length;
    const n = mat[0].length;
    let res = Array(m + n - 1).fill().map(() => []);

    for (let row = 0; row < m; row++) {
        for (let col = 0; col < n; col++) {
            if ((row + col) % 2 === 0) {
                res[row + col].unshift(mat[row][col]);
            } else {
                res[row + col].push(mat[row][col]);
            }
        }
    }
    return res.flat();
};
/*
0      |   1    |    |       2       |    |   3   |    |4|
0,0 -> 0,1 -> 1,0 -> 2,0 -> 1,1 -> 0,2 -> 1,2-> 2,1 -> 2,2
*/
```

# Complexity
- Time: O(m × n) - visit each element once
- Space: O(m × n) - store all elements in the result array before flattening
