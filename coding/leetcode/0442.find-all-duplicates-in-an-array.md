# Find All Duplicates in an Array
[442. Find All Duplicates in an Array](https://leetcode.com/problems/find-all-duplicates-in-an-array/)

Given an integer array nums of length n where all the integers of nums are in the range [1, n] and each integer appears at most twice, return an array of all the integers that appears twice.

# Key insights
- Use the array itself as a hash map (values 1-n map to indices 0 to n-1)
- Mark a number as "seen" by negating the value at index (num - 1)
- If the value at that index is already negative, num appears twice
- Use Math.abs to handle already-negated values

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var findDuplicates = function(nums) {
    const res = [];

    for (let i = 0; i < nums.length; i ++) {
        const num = Math.abs(nums[i]);
        const index = num - 1;

        if (nums[index] < 0) {
            res.push(num);
        } else {
            nums[index] = -nums[index];
        }
    }
    return res;
};
```

# Complexity
**Time:** O(n) - single pass through the array
**Space:** O(1) - modifying the input array in-place (excluding output)
