# Set Matrix Zeroes
[73. Set Matrix Zeroes](https://leetcode.com/problems/set-matrix-zeroes/)

Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0's.

# Key insights
- Use the first row and first column as markers to record which rows/cols need zeroing, with separate flags for the first row/col themselves
- First pass marks zeros in the first row/col; second pass zeros out cells based on those markers
- Final step handles the first row and first column using the boolean flags to avoid overwriting markers prematurely

# Code
```javascript
/**
 * @param {number[][]} matrix
 * @return {void} Do not return anything, modify matrix in-place instead.
 */
var setZeroes = function(matrix) {
    const rows = matrix.length;
    const cols = matrix[0].length;

    if (rows === 0) return;
    let firstColHasZero = false;

    let rowFlag = false;
    let colFlag = false;

    for (let row = 0; row < rows; row++) {
        for (let col = 0; col < cols; col++) {
            if (matrix[row][col] === 0) {
                // FIX: Check both conditions independently instead of else-if
                if (row === 0) {
                    rowFlag = true;
                }
                if (col === 0) {
                    colFlag = true;
                }
                // Only mark in first row/col if not already in first row/col
                if (row !== 0 && col !== 0) {
                    matrix[row][0] = 0;
                    matrix[0][col] = 0;
                }
            }
        }
    }

    for (let row = 1; row < rows; row++) {
        for (let col = 1; col < cols; col++) {
            if (matrix[row][0] === 0 || matrix[0][col] === 0) {
                matrix[row][col] = 0;
            }
        }
    }

    if (rowFlag === true) {
        for (let col = 0; col < cols; col++) matrix[0][col] = 0;
    }

    if (colFlag === true) {
        // FIX: Use matrix[row][0] instead of matrix[row][col] (col is out of scope)
        for (let row = 0; row < rows; row++) matrix[row][0] = 0;
    }
};
```

# Complexity
**Time:** O(m * n)**Space:** O(1)
