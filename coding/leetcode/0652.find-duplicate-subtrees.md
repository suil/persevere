# Find Duplicate Subtrees
[652. Find Duplicate Subtrees](https://leetcode.com/problems/find-duplicate-subtrees/)

Given the root of a binary tree, return all duplicate subtrees.

# Key insights
- Serialize each subtree as a string "val:left:right" using post-order traversal, with '#' for null nodes
- Use a map to count how many times each serialized form appears; add the node to the result on the second occurrence
- This ensures each duplicate subtree is reported exactly once

# Code
```javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {TreeNode[]}
 */
var findDuplicateSubtrees = function(root) {
    const map = new Map();
    const output = []
    findDuplicateSubstreesHelper(root, map, output);
    return output;
};

function findDuplicateSubstreesHelper(node, map, output) {
    if (node === null) {
        return '#';
    }
    
    const key = `${node.val}:${findDuplicateSubstreesHelper(node.left, map, output)}:${findDuplicateSubstreesHelper(node.right, map, output)}`;
    map.set(key, (map.get(key) || 0) + 1);
    if (map.get(key) === 2) {
        output.push(node);
    }
    return key;
}
```

# Complexity
**Time:** O(n^2) -- each node's serialization can be O(n) in length in a skewed tree
**Space:** O(n^2) -- for storing all serialized subtree strings in the map
