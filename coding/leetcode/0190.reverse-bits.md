# Reverse Bits
[190. Reverse Bits](https://leetcode.com/problems/reverse-bits/)

Reverse bits of a given 32 bits signed integer.

# Key insights
- Convert to 32-bit binary string (padded with leading zeros), reverse it, then parse back to integer
- `padStart(32, '0')` ensures all 32 bits are represented before reversing
- Uses string manipulation rather than bit shifting for simplicity

# Code
```javascript
/**
 * @param {number} n - a positive integer
 * @return {number} - a positive integer
 */
var reverseBits = function(n) {
    const bits = n.toString(2).padStart(32, '0').split('').reverse().join('');
    return parseInt(bits, 2);
};
```

# Complexity
**Time:** O(1) -- always processes exactly 32 bits
**Space:** O(1)
