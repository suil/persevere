# Subarray Sum Equals K
[560. Subarray Sum Equals K](https://leetcode.com/problems/subarray-sum-equals-k/)

Given an array of integers nums and an integer k, return the total number of continuous subarrays whose sum equals to k.

Example 1:
Input: nums = [1,1,1], k = 2
Output: 2

Example 2:
Input: nums = [1,2,3], k = 3
Output: 2

# Key insights
* Use prefix sum with hash map to track cumulative sums in O(1) time
* Key insight: if prefixSum[j] - prefixSum[i] = k, then subarray from i+1 to j sums to k
* Initialize map with (0,1) to handle subarrays starting from index 0
* Single pass O(n) solution avoiding nested loops by leveraging prefix sum property

# Code
```javascript
var subarraySum = function(nums, k) {
    let count = 0, sum = 0;
    const map = new Map();
    map.set(0, 1);
    for (const num of nums) {
        sum += num;
        if (map.has(sum - k)) {
            count += map.get(sum - k);
        }
        map.set(sum, (map.get(sum) || 0) + 1);
    }
    return count;
};
```
