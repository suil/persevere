# Bulls and Cows
[299. Bulls and Cows](https://leetcode.com/problems/bulls-and-cows/)

You are playing the Bulls and Cows game with your friend.

# Key insights
- Bulls: digits in correct position (exact matches)
- Cows: correct digits but in wrong positions
- First pass: count bulls and build frequency map of secret digits
- Second pass: count cows using remaining frequencies (after bulls consumed their matches)

# Code
```javascript
/**
 * @param {string} secret
 * @param {string} guess
 * @return {string}
 */
var getHint = function(secret, guess) {
    const freqMap = new Map();
    for (let i = 0; i < secret.length; i++) {
        freqMap.set(secret[i], (freqMap.get(secret[i]) || 0) + 1);
    } 

    let numBull = 0, numCow = 0;
    for (let i = 0; i < guess.length; i++) {
        if (secret[i] === guess[i]) {
            numBull++;
            freqMap.set(guess[i], (freqMap.get(guess[i]) - 1));
        }
    }
    for (let i = 0; i < guess.length; i++) {
        if (secret[i] !== guess[i] && freqMap.get(guess[i]) > 0) {
            numCow++;
            freqMap.set(guess[i], (freqMap.get(guess[i]) - 1));
        }
    }
    return `${numBull}A${numCow}B`;
};
```

# Complexity
**Time:** O(n) - two passes through the strings
**Space:** O(1) - frequency map has at most 10 entries (digits 0-9)
