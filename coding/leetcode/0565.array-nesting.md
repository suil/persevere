# Array Nesting
[565. Array Nesting](https://leetcode.com/problems/array-nesting/)

You are given an integer array nums of length n where nums is a permutation of the numbers in the range [0, n - 1].

# Key insights
- Since nums is a permutation, following indices forms disjoint cycles; mark visited elements as -1 to avoid reprocessing
- Each element belongs to exactly one cycle, so marking ensures O(n) total work across all starts
- Track the maximum cycle length encountered

# Code
```javascript
/**
 * @param {number[]} nums
 * @return {number}
 */
var arrayNesting = function(nums) {
    let max = 0;
    for (let i = 0; i < nums.length; i++) {
        let cnt = 0;
        for (let j = i; nums[j] != -1; ) {
            cnt++;
            let t = nums[j];
            nums[j] = -1; // 标记该位置已经被访问
            j = t;

        }
        max = Math.max(max, cnt);
    }
    return max;
};


```

# Complexity
**Time:** O(n) -- each element is visited at most once due to marking
**Space:** O(1) -- modifies the input array in-place instead of using a separate visited structure
