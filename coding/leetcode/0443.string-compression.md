# String Compression
[443. String Compression](https://leetcode.com/problems/string-compression/)

Given an array of characters chars, compress it using the following algorithm:

# Key insights
- Two pointers: i for reading groups, k for writing compressed result
- For each group of consecutive characters, count the length
- Write the character, then write the count digits (only if count > 1)
- Return k as the new length of the compressed array

# Code
```javascript
/**
 * @param {character[]} chars
 * @return {number}
 */
var compress = function(chars) {
    const len = chars.length;
    let k = 0;
    let i = 0;
    while (i < len) {
        let j = i + 1;
        while (j < chars.length && chars[i] === chars[j]) {
            j++;
        }
        count = j - i;
        // if (j === chars.length) count += 1;
        console.log({ char: chars[i], i, j, count})
        chars[k++] = chars[i];
        if (count > 1) {
            const countStr = `${count}`;
            for (const c of countStr) {
                chars[k++] = c;
            }
        }
        i = j;
    }
    return k;
};
```

# Complexity
**Time:** O(n) - single pass through the array
**Space:** O(1) - compressing in-place
