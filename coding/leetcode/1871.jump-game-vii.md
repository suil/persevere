# Jump Game VII
[1871. Jump Game VII](https://leetcode.com/problems/jump-game-vii/)

You are given a 0-indexed binary string s and two integers minJump and maxJump. In the beginning, you are standing at index 0, which is equal to '0'. You can move from index i to index j if the following conditions are fulfilled:

# Key insights
- BFS from index 0, exploring all reachable '0' positions within jump range
- Track farthest visited index to avoid revisiting already-processed positions
- Start each exploration from max(i + minJump, farthest + 1) to skip redundant checks
- Each index is visited at most once due to farthest pointer optimization

# Code
```javascript
/**
 * @param {string} s
 * @param {number} minJump
 * @param {number} maxJump
 * @return {boolean}
 */
var canReach = function(s, minJump, maxJump) {
    const n = s.length;
    const queue = [0];
    let farthest = 0;

    while (queue.length > 0) {
        const i = queue.shift();
        let start = Math.max(i + minJump, farthest + 1);

        for (let j = start; j <= Math.min(i + maxJump, n - 1); j++) {
            if (s[j] === '0') {
                if (j === n - 1) return true;
                queue.push(j);
            }
        }

        farthest = i + maxJump;
    }

    return false;
};
```

# Complexity
**Time:** O(n) - each index visited at most once due to farthest tracking
**Space:** O(n) - queue can hold up to n indices
