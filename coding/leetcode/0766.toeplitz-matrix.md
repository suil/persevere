# Toeplitz Matrix
[766. Toeplitz Matrix](https://leetcode.com/problems/toeplitz-matrix/)

Given an m x n matrix, return true if the matrix is Toeplitz. Otherwise, return false.

# Key insights
- A Toeplitz matrix requires every diagonal (top-left to bottom-right) to have all equal elements
- Check each diagonal starting from every cell in the first column and first row
- Walk along each diagonal comparing every element to the starting value

# Code
```javascript
/**
 * @param {number[][]} matrix
 * @return {boolean}
 */
var isToeplitzMatrix = function(matrix) {
    for (let row = 0; row < matrix.length; row++) {
        if (!check(matrix, row, 0)) {
            return false;
        }
    }
    
    for (let col = 0; col < matrix[0].length; col++) {
        if (!check(matrix, 0, col)) {
            return false;
        }
    }
    
    return true;
};

function check(matrix, startRow, startCol) {
    let row = startRow;
    let col = startCol;
    
    const startValue = matrix[startRow][startCol];
    while (matrix[row] !== undefined && matrix[row][col] !== undefined) {
        if (matrix[row][col] != startValue) {
            return false;
        }
        row++;
        col++;
    }
    return true;
}
```

# Complexity
**Time:** O(m * n) where m and n are the matrix dimensions**Space:** O(1)
