# Time Needed to Inform All Employees
[1376. Time Needed to Inform All Employees](https://leetcode.com/problems/time-needed-to-inform-all-employees/)

A company has n employees with a unique ID for each employee from 0 to n - 1. The head of the company is the one with headID.

# Key insights
- Builds an adjacency list (manager -> subordinates) from the manager array, then performs DFS from the head
- At each node, the total time is the node's inform time plus the maximum time among all its subordinates
- The answer is the longest path from root to any leaf weighted by inform times

# Code
```javascript
/**
 * @param {number} n
 * @param {number} headID
 * @param {number[]} manager
 * @param {number[]} informTime
 * @return {number}
 */
var numOfMinutes = function(n, headID, manager, informTime) {
    const subordinateMap = new Map();
    for (let i = 0; i < manager.length; i++) {
        if (!subordinateMap.has(manager[i])) {
            subordinateMap.set(manager[i], []);
        }
        subordinateMap.get(manager[i]).push(i);
    }

    return numOfMinutesHelper(subordinateMap, headID, informTime);
};

function numOfMinutesHelper(subordinateMap, id, informTime) {
    if (!subordinateMap.get(id)) {
        return 0
    }

    let max = 0;
    for (const subordinate of subordinateMap.get(id)) {
        max = Math.max(max, informTime[id] + numOfMinutesHelper(subordinateMap, subordinate, informTime));
    }
    return max;
}
```

# Complexity
**Time:** O(n) -- each employee is visited exactly once in the DFS
**Space:** O(n) -- for the subordinate map and recursion stack
