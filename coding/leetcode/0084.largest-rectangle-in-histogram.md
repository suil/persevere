# Largest Rectangle in Histogram
[84. Largest Rectangle in Histogram](https://leetcode.com/problems/largest-rectangle-in-histogram/)

Given an array of integers heights representing the histogram's bar height where the width of each bar is 1, return the area of the largest rectangle in the histogram.

# Key insights
- Monotonic increasing stack of indices; when a shorter bar is encountered, pop and compute area with the popped bar as the height
- Sentinel zeros are added at both ends to ensure all bars get processed
- Width is calculated as the distance between the current index and the new stack top minus one

# Code
```javascript
/**
 * @param {number[]} heights
 * @return {number}
 */
var largestRectangleArea = function(heights) {
    let area = 0;
    let stack = [];
    if (heights.length == 1) return heights[0];

    heights = [0, ...heights, 0];

    for (let i = 0; i < heights.length; i++) {
        while (stack.length > 0 && heights[i] < heights[stack[stack.length - 1]]) {
            const index = stack.pop();
            let width = 1;
            if (stack.length > 0) {
                const nextSmallerIndex = i;
                const prevSmallerIndex = stack[stack.length - 1]
                width = nextSmallerIndex - prevSmallerIndex - 1;
            }
            area = Math.max(area, width * heights[index])
        }
        stack.push(i)
    }
    return area;
};
```

# Complexity
**Time:** O(n)**Space:** O(n)
