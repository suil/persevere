# Shifting Letters
[848. Shifting Letters](https://leetcode.com/problems/shifting-letters/)

You are given a string s of lowercase English letters and an integer array shifts of the same length.

# Key insights
- Each position i is shifted by the sum of shifts[i..n-1]; compute suffix sums from right to left
- Apply each total shift modulo 26 to wrap around the alphabet
- Building suffix sums avoids redundant recomputation of cumulative shifts

# Code
```javascript
/**
 * @param {string} s
 * @param {number[]} shifts
 * @return {string}
 */
var shiftingLetters = function(s, shifts) {
    const totalShifts = [];
    let sum = 0;
    for (let i = shifts.length - 1; i >= 0; i--) {
        sum += shifts[i];
        totalShifts[i] = sum;
    }

    function shift(char, shifts) {
        return String.fromCharCode((char.charCodeAt(0) - 'a'.charCodeAt(0) + shifts) % 26 + 'a'.charCodeAt(0))
        
    }

    const res = s.split('');
    for (let i = 0; i < res.length; i++) {
        res[i] = shift(res[i], totalShifts[i]);
    }
    return res.join('');
};
```

# Complexity
**Time:** O(n)
**Space:** O(n) for the totalShifts and result arrays
