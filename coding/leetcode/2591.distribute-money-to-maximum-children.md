# Distribute Money to Maximum Children
[2591. Distribute Money to Maximum Children](https://leetcode.com/problems/distribute-money-to-maximum-children/)

You are given an integer money denoting the amount of money (in dollars) that you have and another integer children denoting the number of children that you must distribute the money to.

# Code
```javascript
/**
 * @param {number} money
 * @param {number} children
 * @return {number}
 */
var distMoney = function(money, children) {
    if (money < children) return -1;

    let remaining = money - children * 1;

    let count = Math.floor(remaining / 7);

    if (count >= children) {
        if (remaining == 7 * children) return children;
        return children - 1;
    }

    if (count == children - 1) {
        let lastChildMoney = money - 8 * count;
        if (lastChildMoney == 4) {
            return count - 1;
        }
    }

    return count;
};
```
