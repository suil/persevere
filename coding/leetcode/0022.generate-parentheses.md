# Generate Parentheses
[22. Generate Parentheses](https://leetcode.com/problems/generate-parentheses/)

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

# Key insights
* Backtracking: at each step, choose to add `(` if open count < n, or `)` if close count < open count
* The constraint `close < open` ensures we never have more closing parens than opening
* Base case: when both open and close equal n, we have a valid combination

# Code
```javascript
/**
 * @param {number} n
 * @return {string[]}
 */
var generateParenthesis = function(n) {
    function backtrack(open, close, str, output) {
        console.log({open, close, str})
        if (open === n && close === n) {
            output.push(str);
            return;
        }

        if (open < n) {
            backtrack(open + 1, close, str + '(', output);
        }
        if (close < open) {
            backtrack(open, close + 1, str + ')', output);
        }
    }

    const output = [];
    backtrack(0, 0, '', output);
    return output;
};

```

# Complexity
**Time:** O(4^n / sqrt(n)) - The nth Catalan number of valid combinations.
**Space:** O(n) - Recursion depth is 2n.
