# Minimum Area Rectangle
[939. Minimum Area Rectangle](https://leetcode.com/problems/minimum-area-rectangle/)

You are given an array of points in the X-Y plane points where points[i] = [xi, yi].

# Key insights
- Store all points in a nested map for O(1) coordinate lookup
- For each pair of points, treat them as diagonal corners of a potential rectangle
- Check if the other two corners exist (swap x and y coordinates)
- Only consider pairs with different x and y coordinates (non-zero area)

# Code
```javascript
/**
 * @param {number[][]} points
 * @return {number}
 */
var minAreaRect = function(points) {
    const map = new Map();
    for (const point of points) {
        const [x, y] = point;
        if (!map.has(x)) {
            map.set(x, new Map());
        }
        map.get(x).set(y, true);
    }

    let area = Infinity;
    for (let i = 0; i < points.length; i++) {
        for (let j = i + 1; j < points.length; j++) {
            const [x1, y1] = points[i];
            const [x2, y2] = points[j];
            
            if (x1 != x2 && y1 != y2) {
                if (map.get(x1).get(y2) && map.get(x2).get(y1)) {
                    area = Math.min(area, Math.abs(x1 - x2) * Math.abs(y1 - y2));
                }
            }
        }
    }
    return area === Infinity ? 0 : area;
};
```

# Complexity
**Time:** O(n^2) - checking all pairs of points
**Space:** O(n) - for storing points in the map
