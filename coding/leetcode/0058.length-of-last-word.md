# Length of Last Word
[58. Length of Last Word](https://leetcode.com/problems/length-of-last-word/)

Given a string s consisting of words and spaces, return the length of the last word in the string.

# Key insights
- Trim trailing spaces, then split by spaces and return the length of the last element
- Uses `Array.at(-1)` to conveniently access the last word

# Code
```javascript
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLastWord = function(s) {
    s = s.trim();
    let words = s.split(' ');
    return words.at(-1).length;
};
```

# Complexity
**Time:** O(n)**Space:** O(n) for the split array
