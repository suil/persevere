# Greatest Common Divisor of Strings
[1071. Greatest Common Divisor of Strings](https://leetcode.com/problems/greatest-common-divisor-of-strings/)

For two strings s and t, we say "t divides s" if and only if s = t + t + t + ... + t + t (i.e., t is concatenated with itself one or more times).

# Key insights
- If str1 + str2 !== str2 + str1, no common divisor string exists
- When a common divisor exists, its length equals the GCD of the two string lengths
- Uses the Euclidean algorithm to compute the GCD of lengths, then returns that prefix

# Code
```javascript
/**
 * @param {string} str1
 * @param {string} str2
 * @return {string}
 */
var gcdOfStrings = function(str1, str2) {
    function gcd(x, y) {
        if (y === 0) return x;
        return gcd(y, x % y);
    }

    if (str1 + str2 !== str2 + str1) {
        return '';
    }

    const len = gcd(str1.length, str2.length);
    return str1.substring(0, len);
};
```

# Complexity
**Time:** O(m + n) -- string concatenation and comparison dominate
**Space:** O(m + n) -- for the concatenated strings
