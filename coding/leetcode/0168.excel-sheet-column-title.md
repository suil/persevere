# Excel Sheet Column Title
[168. Excel Sheet Column Title](https://leetcode.com/problems/excel-sheet-column-title/)

Given an integer columnNumber, return its corresponding column title as it appears in an Excel sheet.

# Key insights
- This is a base-26 conversion, but 1-indexed (A=1, not A=0), so subtract 1 before taking modulo and dividing
- `(n-1) % 26` gives the current character; `Math.floor((n-1) / 26)` gives the remaining prefix value
- Build the string right-to-left using unshift

# Code
```javascript
/**
 * @param {number} n
 * @return {string}
 */
var convertToTitle = function(n) {
    const codeOfA = 'A'.charCodeAt(0);
    const columnChars = [];
    while (n > 0) {
        columnChars.unshift(String.fromCharCode((n - 1) % 26 + codeOfA));
        n = Math.floor((n - 1) / 26);
    }
    return columnChars.join('');
};

```

# Complexity
**Time:** O(log_26(n))**Space:** O(log_26(n)) for the output characters
