# Integer to English Words
[273. Integer to English Words](https://leetcode.com/problems/integer-to-english-words/)

Convert a non-negative integer num to its English words representation.

Example 1:

Input: num = 123
Output: "One Hundred Twenty Three"
Example 2:

Input: num = 12345
Output: "Twelve Thousand Three Hundred Forty Five"
Example 3:

Input: num = 1234567
Output: "One Million Two Hundred Thirty Four Thousand Five Hundred Sixty Seven"

Constraints:

0 <= num <= 2^31 - 1

# Key insights

- Break the number into **groups of 3 digits** (ones, thousands, millions, billions) and process each group independently
- Numbers 1–19 are irregular (no simple pattern), so they need a direct lookup table
- A helper converts any 3-digit group (1–999) to words by handling hundreds, then delegating the remainder
- Process groups from least significant to most significant, prepending the scale word (Thousand / Million / Billion) after each non-zero group
- Edge case: `num === 0` must be handled upfront since the loop never runs

# Code
```javascript
var numberToWords = function(num) {
    if (num === 0) return 'Zero';

    const ones = ['', 'One', 'Two', 'Three', 'Four', 'Five', 'Six', 'Seven', 'Eight', 'Nine',
                  'Ten', 'Eleven', 'Twelve', 'Thirteen', 'Fourteen', 'Fifteen', 'Sixteen',
                  'Seventeen', 'Eighteen', 'Nineteen'];
    const tens = ['', '', 'Twenty', 'Thirty', 'Forty', 'Fifty', 'Sixty', 'Seventy', 'Eighty', 'Ninety'];
    const scales = ['', 'Thousand', 'Million', 'Billion'];

    function helper(n) {
        if (n === 0) return '';
        if (n < 20) return ones[n] + ' ';
        if (n < 100) return tens[Math.floor(n / 10)] + ' ' + helper(n % 10);
        return ones[Math.floor(n / 100)] + ' Hundred ' + helper(n % 100);
    }

    let result = '';
    for (let i = 0; num > 0; i++) {
        if (num % 1000 !== 0) {
            const scale = scales[i] ? scales[i] + ' ' : '';
            result = helper(num % 1000) + scale + result;
        }
        num = Math.floor(num / 1000);
    }

    return result.trim();
};
```
# Complexity

- **Time**: O(1) — the input is bounded by 2^31 - 1 (at most 4 groups of 3 digits), so work is constant
- **Space**: O(1) — lookup tables and output length are both bounded by a constant
