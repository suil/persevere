# Monotonic Array
[896. Monotonic Array](https://leetcode.com/problems/monotonic-array/)

An array is monotonic if it is either monotone increasing or monotone decreasing.

# Key insights
- Track two boolean flags: could be increasing, could be decreasing
- If any element > next element, it cannot be monotone increasing
- If any element < next element, it cannot be monotone decreasing
- Array is monotonic if at least one flag remains true

# Code
```javascript
/**
 * @param {number[]} A
 * @return {boolean}
 */
var isMonotonic = function(A) {
    let increase = true, decrease = true;
    for(let i = 0; i < A.length - 1; i++) {
        if(A[i] > A[i + 1]) increase = false;
        if(A[i] < A[i + 1]) decrease = false;
    }
    return increase || decrease;
};
```

# Complexity
**Time:** O(n) - single pass through the array
**Space:** O(1) - only using two boolean flags
