# Rotate Image
[48. Rotate Image](https://leetcode.com/problems/rotate-image/)

You are given an n x n 2D matrix representing an image, rotate the image by 90 degrees (clockwise).

# Key insights
- 90-degree clockwise rotation = transpose the matrix, then reverse each row
- Transpose swaps matrix[row][col] with matrix[col][row] for col > row (upper triangle only)
- Both operations are done in-place with no extra space

# Code
```javascript
/**
 * @param {number[][]} matrix
 * @return {void} Do not return anything, modify matrix in-place instead.
 */
var rotate = function(matrix) {
    const n = matrix.length;

    for (let row = 0; row < n; row++) {
        for (let col = row + 1; col < n; col++) {
            [matrix[row][col], matrix[col][row]] = [matrix[col][row], matrix[row][col]];
        }
    }

    for (let row = 0; row < n; row++) {
        matrix[row].reverse();
    }
};
```

# Complexity
**Time:** O(n^2)
**Space:** O(1)
