# Design File System
[1166. Design File System](https://leetcode.com/problems/design-file-system/)

You are asked to design a file system that allows you to create new paths and associate them with different values.

# Key insights
- Uses a Trie where each node represents a path component; children map stores the next path segment
- createPath splits the path, traverses existing nodes, and only creates the final segment (parent path must already exist)
- get traverses the Trie following path components and returns the stored value if the path exists

# Code
```javascript

class TrieNode {
    constructor() {
        this.children = new Map();
        this.isEnd = false;
        this.value = null;
    }
}

var FileSystem = function() {
    this.root = new TrieNode();
};

/** 
 * @param {string} path 
 * @param {number} value
 * @return {boolean}
 */
FileSystem.prototype.createPath = function(path, value) {
    const parts = path.split('/').filter(Boolean);
    let node = this.root;
    for (let i = 0; i < parts.length; i++) {
        const part = parts[i];

        if (!node.children.has(part)) {
            if (i === parts.length - 1) {
                node.children.set(part, new TrieNode())
            } else {
                return false;
            }
        }
        node = node.children.get(part);
    }

    if (node.isEnd) return false;

    node.isEnd = true;
    node.value = value;
    return true;
};

/** 
 * @param {string} path
 * @return {number}
 */
FileSystem.prototype.get = function(path) {
    const parts = path.split('/').filter(Boolean);
    let node = this.root;
    for (const part of parts) {
        if (!node.children.has(part)) return -1;
        node = node.children.get(part);
    }

    if (!node?.isEnd) return -1;
    return node.value;
};

/** 
 * Your FileSystem object will be instantiated and called as such:
 * var obj = new FileSystem()
 * var param_1 = obj.createPath(path,value)
 * var param_2 = obj.get(path)
 */
```

# Complexity
**Time:** O(P) per operation -- where P is the number of components in the path
**Space:** O(total path components) -- one Trie node per unique path segment
