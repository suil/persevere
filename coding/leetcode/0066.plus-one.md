# Plus One
[66. Plus One](https://leetcode.com/problems/plus-one/)

You are given a large integer represented as an integer array digits, where each digits[i] is the ith digit of the integer. The digits are ordered from most significant to least significant in left-to-right order. The large integer does not contain any leading 0's.

# Key insights
- Iterate from the last digit backward; if a digit is less than 9, increment it and return immediately (no carry)
- If the digit is 9, set it to 0 and continue propagating the carry leftward
- If all digits were 9 (loop completes), prepend 1 to handle overflow (e.g., 999 -> 1000)

# Code
```javascript
/**
 * @param {number[]} digits
 * @return {number[]}
 */
var plusOne = function(digits) {
    for (let i = digits.length - 1; i >= 0; i--) {
        if (digits[i] < 9) {
            digits[i]++;
            return digits;
        }
        digits[i] = 0;
    }

    digits.unshift(1)
    return digits;
};
```

# Complexity
**Time:** O(n)**Space:** O(1) amortized (O(n) only when overflow requires a new digit)
