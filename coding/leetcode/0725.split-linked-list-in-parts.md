# Split Linked List in Parts
[725. Split Linked List in Parts](https://leetcode.com/problems/split-linked-list-in-parts/)

Given the head of a singly linked list and an integer k, split the linked list into k consecutive linked list parts.

# Key insights
- First pass counts total length, then compute averageLen = floor(len/k) and remainder = len % k
- The first `remainder` parts get averageLen + 1 nodes, the rest get averageLen nodes, distributing extra nodes evenly
- Sever each part by setting prev.next = null after traversing the appropriate number of nodes

# Code
```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} root
 * @param {number} k
 * @return {ListNode[]}
 */
var splitListToParts = function(root, k) {
    let len = 0
    let curr = root
    while (curr) {
        len++
        curr = curr.next
    }

    const averageLen = Math.floor(len / k)
    let remainder = len % k
    const res = new Array(k)

    for (let i = 0; i < k; i++) {
        res[i] = root

        let realLen = (remainder > 0) ? averageLen + 1 : averageLen
        let prev = null
        remainder--

        while (realLen > 0 && root) {
            prev = root
            root = root.next
            realLen--
        }

        if (prev) {
            prev.next = null
        }
    }

    return res;
};
```

# Complexity
**Time:** O(n + k) where n is the list length
**Space:** O(k) for the result array
