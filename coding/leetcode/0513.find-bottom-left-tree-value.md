# Find Bottom Left Tree Value
[513. Find Bottom Left Tree Value](https://leetcode.com/problems/find-bottom-left-tree-value/)

Given the root of a binary tree, return the leftmost value in the last row of the tree.

Example 1:
        2
       / \
      1   3
Input: root = [2,1,3]
Output: 1

Example 2:
             1
            / \
           2   3
          /   / \
         4   5   6
            /
           7
Input: root = [1,2,3,4,null,5,6,null,null,7]
Output: 7

# Key insights
* Use BFS (level-order traversal) with queue to process tree level by level
* Track leftmost node of current level by capturing first element in queue
* Build next level's queue while processing current level to maintain structure
* Return last recorded leftmost value when no more levels exist

# Code
```javascript
var findBottomLeftValue = function(root) {
    let queue = [root];
    let bottomLeftNode = root.val;
    
    while (queue.length > 0) {
        bottomLeftNode = queue[0];
        const nextQueue = [];
        for (const node of queue) {
            if (node.left) { nextQueue.push(node.left); }
            if (node.right) { nextQueue.push(node.right); }
        }
        queue = nextQueue;
    }
    return bottomLeftNode.val
};
```

# Complexity
**Time:** O(n) - visits each node in the tree exactly once during the level-order traversal
**Space:** O(w) - queue stores at most all nodes at the widest level of the tree, where w is the maximum width
