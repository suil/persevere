# Uncrossed Lines
[1035. Uncrossed Lines](https://leetcode.com/problems/uncrossed-lines/)

You are given two integer arrays nums1 and nums2. We write the integers of nums1 and nums2 (in the order they are given) on two separate horizontal lines.

# Key insights
- This is equivalent to the Longest Common Subsequence (LCS) problem
- Uses a 2D DP table where dp[i][j] represents the max uncrossed lines between nums1[0..i-1] and nums2[0..j-1]
- When elements match, extend the diagonal; otherwise, take the max of skipping one element from either array

# Code
```javascript
/**
 * @param {number[]} nums1
 * @param {number[]} nums2
 * @return {number}
 */
var maxUncrossedLines = function(nums1, nums2) {
    // function backtrack(index1, index2) {
    //     if (index1 >= nums1.length || index2 >= nums2.length) return 0;

    //     if (nums1[index1] === nums2[index2]) {
    //         return 1 + backtrack(index1 + 1, index2 + 2);
    //     }

    //     return Math.max(
    //         backtrack(index1 + 1, index2),
    //         backtrack(index1, index2 + 1)
    //     );
    // }

    // return backtrack(0, 0);

    const dp = Array(nums1.length + 1).fill().map(() => Array(nums2.length + 1).fill(0));

    for (let i = 1; i <= nums1.length; i++) {
        for (let j = 1; j <= nums2.length; j++) {
            if (nums1[i - 1] === nums2[j - 1]) {
                dp[i][j] = 1 + dp[i - 1][j - 1];
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[nums1.length][nums2.length];
};
```

# Complexity
**Time:** O(m * n) -- where m and n are the lengths of nums1 and nums2
**Space:** O(m * n) -- for the 2D DP table
