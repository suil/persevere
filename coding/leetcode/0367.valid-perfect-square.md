# Valid Perfect Square
[367. Valid Perfect Square](https://leetcode.com/problems/valid-perfect-square/)

Binary Search:

Math:
The series of square num：1,4,9,16,..

The gap between square numbers is ：3,5,7,...

# Key insights
* Binary search eliminates half the search space each iteration for O(log n) efficiency
* Search range from 0 to num, comparing mid² against target to narrow bounds
* Avoid integer overflow by using (hi - lo) / 2 instead of (hi + lo) / 2
* Early termination when num < 2 handles edge cases of 0 and 1

# Code
```javascript
var isPerfectSquare = function(num) {
    if (num < 2) { return true; }
    let lo = 0, hi = num;
    while (lo <= hi) {
        const mid = Math.floor(lo + (hi - lo) / 2);
        const squre = mid * mid;
        if (squre === num) { return true; }
        if (squre > num) {
            hi = mid - 1;
        } else {
            lo = mid + 1;
        }
    }
    return false;
};
```

# Complexity
**Solution 1:**
**Time:** O(log n) - binary search halves the search space in each iteration
**Space:** O(1) - uses only a constant amount of extra variables

**Solution 2:**
**Time:** O(√n) - subtracts consecutive odd numbers until reaching zero, requiring √n iterations
**Space:** O(1) - uses only a constant amount of extra variables
