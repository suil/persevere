# Valid Number
[65. Valid Number](https://leetcode.com/problems/valid-number/)

Given a string s, return whether s is a valid number.

# Key insights
- Single regex validates the entire number format: optional sign, integer/decimal part, optional exponent with optional sign and digits
- The decimal part handles three cases: digits.digits, digits., and .digits
- Trims whitespace before testing

# Code
```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var isNumber = function(s) {
    return /^[+-]?((\d+(\.\d+)?|(\d+\.)|(\.\d+)))([eE][+-]?\d+)?$/.test(s.trim());
};

```

# Complexity
**Time:** O(n)**Space:** O(1)
