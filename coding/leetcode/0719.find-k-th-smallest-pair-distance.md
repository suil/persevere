# Find K-th Smallest Pair Distance
[719. Find K-th Smallest Pair Distance](https://leetcode.com/problems/find-k-th-smallest-pair-distance/)

The distance of a pair of integers a and b is defined as the absolute difference between a and b.

# Key insights
- Binary search on the answer (distance value), not on indices; the search space is [0, max - min]
- For each candidate distance, use a sliding window on the sorted array to count how many pairs have distance <= candidate
- Sorting enables the sliding window: for each right pointer, advance left until nums[right] - nums[left] <= distance
- The count of valid pairs for a given right index is simply (right - left), since all elements in that range form valid pairs with right

# Code
```javascript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number}
 */
var smallestDistancePair = function(nums, k) {
    nums.sort((a, b) => a - b);

    function canFindPair(distance) {
        let count = 0;
        let left = 0;
        for (let right = 0; right < nums.length; right++) {
            while (left < nums.length && nums[right] - nums[left] > distance) {
                left++;
            }
            count += right - left; // pair ending at right
        }
        return count >= k;
    }

    let left = 0;
    let right = Math.abs(nums.at(-1) - nums[0]);

    let res = right;
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (canFindPair(mid)) {
            right = mid - 1;
            res = mid;
        } else {
            left = mid + 1;
        }
    }
    return res;
};
```

# Complexity
**Time:** O(n log n + n log W) where W is the max pair distance (sorting + binary search with linear scan)
**Space:** O(1) extra space (ignoring sort)
