# Monotone Increasing Digits
[738. Monotone Increasing Digits](https://leetcode.com/problems/monotone-increasing-digits/)

An integer has monotone increasing digits if and only if each pair of adjacent digits x and y satisfy x <= y.

# Key insights
- Traverse digits from right to left; when a digit is greater than its right neighbor, decrement it and set all digits to its right to 9
- Setting trailing digits to 9 maximizes the result while ensuring the number stays <= n
- Right-to-left traversal handles cascading decrements (e.g., 332 -> 299)

# Code
```javascript
/**
 * @param {number} n
 * @return {number}
 */
var monotoneIncreasingDigits = function(n) {
    const nums = n.toString().split('');
    for (let i = nums.length - 2; i >= 0; i--) {
        if (nums[i] > nums[i + 1]) {
            nums[i]--;
            for (let j = i + 1; j < nums.length; j++) {
                nums[j] = 9;
            }
        }
    }
    return Number(nums.join(''));
};
```

# Complexity
**Time:** O(d^2) where d is the number of digits (outer loop * inner fill loop, though d <= 10)
**Space:** O(d) for the digit array
