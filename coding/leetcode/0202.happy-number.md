# Happy Number
[202. Happy Number](https://leetcode.com/problems/happy-number/)

Write an algorithm to determine if a number n is happy.

# Key insights
- Repeatedly replace the number with the sum of squares of its digits
- Use a Set to detect cycles: if a number repeats, it will never reach 1
- Return true when the number becomes 1, false when a cycle is detected

# Code
```javascript
/**
 * @param {number} n
 * @return {boolean}
 */
var isHappy = function(n) {
    const set = new Set();
    let num = n;

    while (true) {
        let newNum = 0;
        while (num > 0) {
            newNum += (num % 10) ** 2;
            num = Math.floor(num / 10);
        }
        if (set.has(newNum)) return false;
        if (newNum === 1) return true;
        set.add(newNum);
        num = newNum;
    }
    return false;
};
```

# Complexity
**Time:** O(log n) per step, bounded total iterations
**Space:** O(log n) for the Set of seen numbers
