# Sparse Matrix Multiplication
[311. Sparse Matrix Multiplication](https://leetcode.com/problems/sparse-matrix-multiplication/)

Given two sparse matrices mat1 of size m x k and mat2 of size k x n, return the result of mat1 x mat2. You may assume that multiplication is always possible.

Example 1:

[1, 0, 0]
[-1,0, 3]

[7,0,0],
[0,0,0],
[0,0,1]

Input: mat1 = [[1,0,0],[-1,0,3]], mat2 = [[7,0,0],[0,0,0],[0,0,1]]
Output: [[7,0,0],[-7,0,3]]

Example 2:
Input: mat1 = [[0]], mat2 = [[0]]
Output: [[0]]

# Key insights
* Standard matrix multiplication using three nested loops (O(n³) approach)
* Multiply row elements from mat1 with column elements from mat2
* Does not optimize for sparse matrices - treats all zero values equally
* Space complexity O(m×p) for result matrix where m=rows, p=columns

# Code
```javascript
var multiply = function(mat1, mat2) {
    const res = [];
    for (let row = 0; row < mat1.length; row++) {
        res.push([]);
        for (let col = 0; col < mat2[0].length; col++) {
            let sum = 0;
            for (let i = 0; i < mat2.length; i++) {
                sum += mat1[row][i] * mat2[i][col];
            }
            res[row][col] = sum
        }
    }
    return res;
};
```
